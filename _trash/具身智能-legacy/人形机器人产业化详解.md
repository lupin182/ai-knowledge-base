# 人形机器人产业化详解（2026）

> 聚焦硬件、供应链、量产和商业化。不重复 VLA 模型内容，只讲"本体"与"落地"。

## 一、为什么是人形？

### 本体形态之争

```
轮式 + 机械臂 (轮式双臂)      双足人形
──────────────────────        ──────────────────
低成本 ($5-15K)                高成本 ($30-100K+)
结构化环境优势                  非结构化环境优势
电池续航长                      电池续航短 (1-4h)
不能上下楼/跨越障碍             几乎全地形
代表: Galbot G1, Stretch 3     代表: Figure, Optimus, Neo, Digit
     UR5 on AMR                Unitree H1/G1
```

**人形论据**：
- **世界为人类设计**：门把手高度、楼梯、工具都适配人体尺度
- **数据形态统一**：海量人类视频（YouTube, 第一视角）可用于预训练
- **通用性溢价**：单一形态覆盖多场景，长期规模效应

**反对论据**：
- 双足控制难度远高于轮式
- 电池密度不足，续航差
- 安全顾虑大（摔倒、碰撞）

### 形态谱系（2026 年主流方案）

| 形态 | 代表 | 定位 |
|------|-----|------|
| **全尺寸双足人形** | Optimus, Figure 02/03, Apollo, H1 | 通用，B + 未来 C |
| **轻量双足人形** | Unitree G1, Neo, PM01, STAR1 | 科研 + 消费 |
| **轮式双足（轮腿）** | Agility Digit (鸟腿), Boston Atlas (mixed) | 物流专用 |
| **轮式人形上半身** | Galbot G1, Halodi EVE | 家庭 / 服务 |
| **固定底座双臂** | ALOHA, Franka + 另一台 | 研究 / 工业 |

## 二、硬件关键模块

### 2.1 总体 BOM 与成本分布

以 2025 年中端人形（Figure 02 / Unitree G1 高配）为例：

```
总成本 $30-50K 分布:
├── 执行器（关节电机 + 减速器）    40-50%  ($15-25K)
├── 灵巧手                       10-15%  ($3-7K)
├── 传感器（IMU, RGB-D, 触觉）     5-10%   ($2-5K)
├── 计算平台 (Jetson Thor/Orin)    5-8%    ($2-4K)
├── 结构件 (碳纤维 + 铝合金)       10%     ($3-5K)
├── 电池                         5-8%    ($2-3K)
├── 电源管理 / 线束 / 外壳        10%     ($3-5K)
└── 总装 / 测试 / 认证            5-10%
```

**关键观察**：**执行器占成本近半**。批量化降本的核心是关节电机规模生产。

### 2.2 关节电机技术路线

```
路线 A: 准直驱 BLDC + 谐波减速器
  特点: 高精度、高刚度、背驱好
  代表: Franka Panda, Figure, Optimus
  成本: 单关节 $500-2000

路线 B: 传统伺服 + 行星减速器
  特点: 成本低、国产化
  代表: 国产人形常用
  成本: 单关节 $200-600

路线 C: 腱绳驱动 (Tendon-Driven)
  特点: 远程布置电机、轻量末端
  代表: Shadow Hand, 部分灵巧手
  成本: 高 (机械设计复杂)

路线 D: 准直驱（低减速比）
  特点: 反向可驱、适合 locomotion
  代表: Unitree A1/G1 四足, 部分人形腿部
  成本: 中
```

**2026 趋势**：人形机器人腿部用**低减速比 + 大扭矩电机**（适合动态），手臂用**谐波减速器**（适合精度）。

**关键技术**：
- **非降维全驱动**：每个关节独立电机（Optimus Gen 2、Figure v2 方向）
- **触觉集成**：GelSight (视触觉)、Digit 360 (Meta)、BioTac
- **指尖力控**：实现抓握力调节，不只是开合
- **手掌设计**：刚柔复合、掌心曲面、指腹贴合

**2026 主要难点**：
1. 触觉阵列 + 关节电机 + 腱绳 的耦合设计
2. 寿命问题：腱绳磨损、电机过热
3. 标定复杂性：每只手出厂需独立标定

### 2.4 传感器配置

```
典型配置:
├── 头部
│   ├── 2-3 路 RGB 相机 (stereo + 独立)
│   ├── 1 路深度相机 (ToF / 结构光)
│   └── 麦克风阵列 (可选)
├── 腕部
│   └── 2 路 RGB 相机 (per arm, 近距操作)
├── 手部
│   ├── 关节编码器
│   └── 指尖触觉 (可选)
├── 本体
│   ├── IMU (通常 2 个: base + torso)
│   ├── 关节力矩传感器 (或电流估算)
│   └── 足底压力 (双足)
└── (可选) LiDAR / 毫米波

2025 主流 RGB-D 相机:
  Intel RealSense D435/D455/D457
  Orbbec Gemini 335
  ZED X

2025 主流触觉:
  GelSight Mini / Digit 360 / Paxini / XELA
```

## 四、商业化路径分析

### 4.1 工业场景 (2025-2027 主战场)

| 场景 | 典型任务 | 人形 vs 传统方案 | 已部署案例 |
|------|---------|----------------|----------|
| **汽车工厂** | 零件搬运、分拣、插件 | 优势: 混线灵活性 | Figure@BMW, Apptronik@奔驰, 优必选@东风 |
| **物流仓库** | 拣货、装箱、码垛 | 优势: 不改造仓库 | Digit@GXO/Amazon, 京东自研 |
| **电子厂** | SMT 辅助、包装 | 优势: 处理异形 | 富士康@UBTECH |
| **电池工厂** | 电芯装配 | 优势: 高精度操作 | Tesla Optimus@Gigafactory |

**典型 ROI 计算**：
```
传统工业机器人 ROI (6-18 月)
人形机器人 ROI (24-48 月, 2025 年)
→ 2027-2028 降至 18-30 月，进入主流采购决策区
```

### 4.2 服务场景 (2026-2028)

- 酒店迎宾 / 送物
- 展厅导览
- 康复医疗辅助
- 餐饮送餐（已有特调版本）

### 4.3 家庭场景 (2027+ 远期)

家用的三道坎：
1. **安全**：跌倒、误伤儿童、宠物
2. **数据**：长尾家庭场景无限多样
3. **成本**：>$10K 家庭接受度极低

**2026 典型家庭早期产品**：
- 1X Neo Beta（订阅制）
- Figure 03 演示（非量产）

### 4.4 B 端 vs C 端成本敏感度

```
B 端 (ROI 驱动):
  接受 $50-200K
  接受 2-5 年回本
  主要成本: 人工替代 ($30-80K/人年)

C 端 (愿付价 + 替代品):
  接受 $5-20K
  对标: 汽车第 2 辆/高端家电/保姆
  2028-2030 可能拐点
```

## 七、风险与争议

### 7.1 技术风险

- **安全事故**：一次严重事故可能击穿信任（类比 Cruise Robotaxi）
- **大脑不可用**：VLA 模型长尾泛化失败率仍高
- **硬件可靠性**：MTBF（无故障时间）短，维护成本高

### 7.2 商业风险

- **ROI 不达预期**：人形成本下降慢于预期
- **资本收缩**：2026-2027 若无明显 PMF，估值可能腰斩
- **监管政策**：工伤责任、数据隐私（家用场景）

### 7.3 地缘政治

- 芯片出口管制（Jetson Thor 对华）
- 机器人关键零部件供应链
- 数据本地化要求

## 八、给从业者的判断

1. **2026 是"B 端规模化验证"之年**，不是家庭普及元年。
2. **硬件成本下降快于大脑能力提升**。2026 年瓶颈在"大脑"，不在"身体"。
3. **垂直整合短期领跑，水平分层长期胜出**。但"分层"何时胜出仍不清晰。
4. **中国本体厂规模化优势明显，美国模型公司质量优势明显**。格局未定。
5. **真正的护城河是数据 + 用户反馈飞轮，而非模型或硬件**。
6. **2027 年前无真正的 C 端"iPhone 时刻"**。
