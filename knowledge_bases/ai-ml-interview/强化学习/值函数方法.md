# 值函数方法 (Value-Based Methods)

通过估计值函数 $V(s)$ 或 $Q(s,a)$，间接推导最优策略。

## 1. 动态规划 (DP)

前提：**已知完整的 MDP 模型**（转移概率和奖励函数）。

### 策略迭代 (Policy Iteration)

交替进行两步，直到收敛：

1. **策略评估**：给定策略 $\pi$，解 Bellman 期望方程得到 $V^\pi$
2. **策略改进**：对每个状态贪心更新 $\pi'(s) = \arg\max_a Q^\pi(s,a)$

```python
# 策略评估
while not converged:
    for s in states:
        V[s] = sum(pi[s][a] * (R[s][a] + gamma * sum(P[s][a][s_next] * V[s_next]
               for s_next in states)) for a in actions)

# 策略改进
for s in states:
    pi[s] = argmax_a(sum(P[s][a][s_next] * (R[s][a] + gamma * V[s_next])
            for s_next in states))
```

### 值迭代 (Value Iteration)

直接迭代 Bellman 最优方程：

$$V_{k+1}(s) = \max_a \left[ R(s,a) + \gamma \sum_{s'} P(s' \mid s,a) V_k(s') \right]$$

收敛后从 $V^*$ 导出最优策略。比策略迭代更简洁，但每步都在做 max 操作。

## 2. Monte Carlo 方法

**不需要环境模型**，通过采样完整轨迹来估计值函数。

$$V(s) \approx \frac{1}{N} \sum_{i=1}^{N} G_t^{(i)}$$

### First-Visit vs Every-Visit MC

| 类型 | 做法 | 特点 |
|------|------|------|
| First-Visit | 只用每条轨迹中状态 $s$ 第一次出现时的回报 | 无偏估计 |
| Every-Visit | 用轨迹中状态 $s$ 每次出现时的回报 | 有偏但方差更低 |

### MC 的优缺点

- **优点**：无偏、不需要环境模型、不受马尔可夫假设约束
- **缺点**：必须等到回合结束才能更新（高方差）、不适用于连续任务

## 3. 时序差分学习 (TD Learning)

结合 MC 的无模型采样和 DP 的自举 (Bootstrapping)：

$$V(s_t) \leftarrow V(s_t) + \alpha \left[ \underbrace{r_{t+1} + \gamma V(s_{t+1})}_{\text{TD Target}} - V(s_t) \right]$$

其中 $\delta_t = r_{t+1} + \gamma V(s_{t+1}) - V(s_t)$ 称为 **TD 误差**。

### MC vs TD 的核心区别

| 维度 | MC | TD |
|------|-----|-----|
| 更新时机 | 回合结束 | 每一步 |
| 偏差 | 无偏 | 有偏（因为自举） |
| 方差 | 高 | 低 |
| 收敛性 | 收敛到 $V^\pi$ | 收敛到 $V^\pi$（线性函数逼近时） |
| 适用场景 | 回合制任务 | 连续任务也可用 |

### TD(n) 与 TD(λ)

- **TD(0)**：只看一步 $r_{t+1} + \gamma V(s_{t+1})$
- **TD(n)**：看 $n$ 步回报 $G_t^{(n)} = \sum_{k=0}^{n-1} \gamma^k r_{t+k+1} + \gamma^n V(s_{t+n})$
- **TD(λ)**：对所有 $n$ 步回报的指数加权平均，$\lambda=0$ 退化为 TD(0)，$\lambda=1$ 退化为 MC

$$G_t^\lambda = (1-\lambda) \sum_{n=1}^{\infty} \lambda^{n-1} G_t^{(n)}$$

实践中通过**资格迹 (Eligibility Traces)** 高效实现。

## 4. Q-Learning

最经典的 Off-Policy TD 控制算法（Watkins, 1989）。

$$Q(s_t, a_t) \leftarrow Q(s_t, a_t) + \alpha \left[ r_{t+1} + \gamma \max_{a'} Q(s_{t+1}, a') - Q(s_t, a_t) \right]$$

**关键特征**：
- 更新时用 $\max_{a'} Q(s_{t+1}, a')$（贪心），而行为策略可以是 $\epsilon$-greedy
- 因此是 **Off-Policy**：行为策略 ≠ 目标策略
- 在表格情况下，保证收敛到 $Q^*$

### SARSA

On-Policy 版本的 TD 控制：

$$Q(s_t, a_t) \leftarrow Q(s_t, a_t) + \alpha \left[ r_{t+1} + \gamma Q(s_{t+1}, a_{t+1}) - Q(s_t, a_t) \right]$$

用 **实际执行的** $a_{t+1}$ 而非 $\max$。因此更保守，在有悬崖的环境中更安全。

### Q-Learning vs SARSA

| 维度 | Q-Learning | SARSA |
|------|-----------|-------|
| 类型 | Off-Policy | On-Policy |
| 目标 | $\max_{a'} Q(s', a')$ | $Q(s', a')$，$a'$ 是实际动作 |
| 收敛到 | 最优策略 | 当前 $\epsilon$-greedy 策略 |
| 特点 | 更激进 | 更保守、更安全 |

## 5. DQN (Deep Q-Network)

用神经网络逼近 $Q(s,a;\theta)$，突破表格方法的维度限制（Mnih et al., 2015）。

### 核心创新

#### 1. Experience Replay

将经验 $(s, a, r, s')$ 存入 Replay Buffer，训练时随机采样 mini-batch：

- 打破样本之间的时序相关性
- 提高数据利用效率
- Off-Policy 天然支持

#### 2. Target Network

使用一个延迟更新的目标网络 $\theta^-$ 计算 TD Target：

$$L(\theta) = \mathbb{E}\left[\left(r + \gamma \max_{a'} Q(s', a'; \theta^-) - Q(s, a; \theta)\right)^2\right]$$

每隔 $C$ 步同步 $\theta^- \leftarrow \theta$。作用：稳定训练，避免"追逐移动目标"。

### DQN 训练流程

```python
# 初始化
Q_network = QNetwork(theta)
Q_target = QNetwork(theta_minus)  # 拷贝参数
replay_buffer = ReplayBuffer(capacity=1_000_000)

for episode in range(num_episodes):
    s = env.reset()
    while not done:
        # epsilon-greedy 选动作
        a = epsilon_greedy(Q_network, s, epsilon)
        s_next, r, done = env.step(a)
        replay_buffer.push(s, a, r, s_next, done)

        # 采样训练
        batch = replay_buffer.sample(batch_size=32)
        td_target = batch.r + gamma * Q_target(batch.s_next).max(dim=1) * (1 - batch.done)
        loss = MSE(Q_network(batch.s, batch.a), td_target)
        optimizer.step(loss)

        # 定期同步目标网络
        if step % C == 0:
            Q_target.load_state_dict(Q_network.state_dict())
```

## 6. DQN 改进系列

| 改进 | 解决的问题 | 核心思想 |
|------|-----------|---------|
| **Double DQN** | Q 值过高估计 | 用 Q 网络选动作，目标网络评估：$Q(s', \arg\max_{a'} Q(s',a';\theta); \theta^-)$ |
| **Dueling DQN** | 状态和优势混在一起 | 分离 $Q = V(s) + A(s,a)$，共享底层特征 |
| **Prioritized ER** | 均匀采样低效 | TD 误差大的样本优先采样 |
| **Noisy Net** | $\epsilon$-greedy 探索粗糙 | 在网络参数中加噪声，学习探索策略 |
| **Distributional DQN (C51)** | 只学均值 Q 信息不够 | 学习回报的完整分布 |
| **Rainbow** | 各改进正交互补 | 集成以上所有改进 |

### Double DQN 的直觉

标准 DQN 的 $\max_{a'} Q(s', a')$ 同时用 Q 网络选动作和评估价值，导致系统性高估。Double DQN 解耦这两步：

$$Y = r + \gamma Q(s', \arg\max_{a'} Q(s', a'; \theta); \theta^-)$$

- $\theta$：选动作
- $\theta^-$：评估价值
