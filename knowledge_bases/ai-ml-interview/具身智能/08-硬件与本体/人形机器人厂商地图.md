# 人形机器人厂商地图（2026）

> 截至 2026 年 4 月的主流人形机器人本体厂商，按地域与定位梳理。包含主打路线、代表产品、技术特色、商业化进展。数字均为官方披露或据报道估算。

## 一、美国阵营

### Tesla Optimus

| 项 | 内容 |
|---|---|
| **主打路线** | 垂直整合（电机/电池/AI 芯片自产）+ 超大规模自产，对标 Model 3 |
| **代表产品** | Optimus Gen 2（2024.10）、Gen 3（2026 预期） |
| **身高/体重** | 173cm / 57kg（Gen 2） |
| **自由度** | 约 28（不含手）；手 11 DOF × 2（Gen 2）；Gen 3 预计手部 22 主动 DOF |
| **芯片** | Tesla FSD 自研 SoC |
| **技术特色** | FSD 视觉堆栈复用；供应链垂直整合；电机/减速器自产 |
| **商业化** | 2026 年内部产能目标 5,000-10,000 台（工厂员工/Gigafactory 试点）；2027 计划对外销售；Gen 3 成本目标 \$20-30K |

### Figure AI

| 项 | 内容 |
|---|---|
| **主打路线** | 垂直整合（硬件 + Helix 模型），B 端商业落地 |
| **代表产品** | Figure 01（2023 概念）、Figure 02（2024.08）、Figure 03（2025.10） |
| **身高/体重** | 168cm / 70kg（Figure 02） |
| **芯片** | NVIDIA Jetson Orin（02）；Jetson Thor（03 预计） |
| **技术特色** | Helix VLA（7B VLM @ 9Hz + 80M Transformer @ 200Hz，双频双系统）；Figure Hand v2（16 DoF）；双机协作 |
| **商业化** | BMW Spartanburg 工厂持续部署（2024-）；B2B 订阅模式约 \$3-5K/月；融资 \$675M（2024，估值 \$2.6B）；从 OpenAI 合作转向自研 |

### 1X Technologies（挪威/美国）

| 项 | 内容 |
|---|---|

### Agility Robotics（Digit）

| 项 | 内容 |
|---|---|
| **主打路线** | 物流专用双足，"先钻透一个场景" |

### Apptronik（Apollo）

### Boston Dynamics（电动 Atlas）

## 四、战略定位的三角

```
         垂直整合 / 自研全栈
         Figure · Tesla · 1X
              /\
             /  \
            /    \
           /      \
模型 / 平台 ───── 本体 / 制造
π₀ · NVIDIA       Unitree · 傅利叶
DeepMind          EngineAI · 智元本体
Skild             UBTECH 工厂版
```

- **垂直整合（Figure/Tesla/1X）**：短期商业化领跑，数据闭环强，但资本密集。
- **模型 + 平台（π₀/NVIDIA/DeepMind）**：轻资产，覆盖多本体，但依赖硬件合作方。
- **本体规模化（Unitree/傅利叶/EngineAI）**：硬件成本与产能优势，溢价空间低，大脑依赖第三方。

**2026 年判断**：
- 工业 B 端：垂直整合（Figure/Tesla/Apptronik+DeepMind）占先
- 研究与早期部署：本体厂（Unitree/傅利叶）+ 第三方 VLA（GR00T/π₀）组合
- 家庭 C 端：1X、Figure 03、智元灵犀 X2 是主要 Beta 候选
