# 08 · 硬件与本体

> 具身智能的"身体"：从整机形态、关节执行器、灵巧手到触觉传感器与 BOM 成本。本目录聚焦 2025-2026 年人形机器人硬件的产业化现状。

## 2026 硬件发展现状

### 三个关键信号

1. **BOM 成本跌至 \$20-35K 区间**
   - 2024 年：主流双足人形 BOM \$50-100K，个别研究平台（老 Atlas、Shadow 手整套）\$200K+
   - 2025 年：量产初期降至 \$30-50K（Figure 02 对外 ~\$100K，Unitree G1 起步 ¥9.9 万≈\$14K）
   - 2026 年：**头部厂商批量订单进入 \$20-35K 区间**，对应 B 端 ROI 2-4 年回本，工业场景可谈
   - 拐点驱动：关节电机规模化、中国供应链替代、结构件减重（碳纤+塑化）

2. **灵巧手从研究玩具转向量产部件**
   - 2024 年主流仍是 Allegro / LEAP / Shadow 这些研究硬件，单价 \$2K-150K 不等
   - 2025 年 Tesla Optimus Hand Gen 2（22 主动自由度）、Figure Hand v2、因时 RH56DFX（¥3 万≈\$4K）批量出货
   - **非降维全驱动**（每关节一个独立电机，无耦合简化）成为高端标配
   - 触觉、力控从"可选"变为"标配"的过渡期

3. **触觉集成开始普及**
   - GelSight（视触觉）、Meta Digit 360、PaXini 指端阵列触觉传感器进入量产手
   - 2025-2026 的工程重点：**触觉数据进入 VLA 训练主回路**（此前视觉主导，触觉仅做安全兜底）
   - 工程难点仍在：阵列传感器的寿命、带宽、标定复杂度

### 本体形态的分化

```
全尺寸双足人形       轻量双足（科研/消费）      轮式人形上半身
Optimus, Figure,      Unitree G1, Neo,          Galbot G1, Halodi
Apollo, H1, Atlas     PM01, STAR1               EVE
│ 通用 + 工业         │ 研究 / 消费 Beta        │ 结构化环境 / 家庭
│ \$30-100K+          │ \$10-20K                │ \$15-40K
```

**判断**：2026 年工业与物流部署仍以全尺寸双足为主，消费 Beta 以轻量双足 + 轮式人形上半身并行。
