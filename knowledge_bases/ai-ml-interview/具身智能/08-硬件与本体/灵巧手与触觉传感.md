# 灵巧手与触觉传感（2026）

> 灵巧手是人形机器人"末端执行器"的核心组件，2025-2026 年从研究玩具快速过渡到量产部件。本文梳理研究用/量产灵巧手、驱动方式、触觉传感器与工程难点。

## 一、研究用灵巧手（学术主流）

| 型号 | 单位 | 自由度（主动/总） | 驱动 | 触觉 | 单价（估算） | 开源 |
|-----|------|----------------|------|-----|-----------|------|
| **Allegro Hand v4** | Wonik (KR) | 16 / 16 | 电机直驱（每指 4 个关节独立） | 无 | ~\$15K | ❌ |
| **LEAP Hand** | CMU | 16 / 16 | 电机直驱（Dynamixel 舵机） | 无 | ~\$2K（自装）/ ~\$4K（成品） | ✅ 开源硬件 |
| **Shadow Hand** | Shadow Robot (UK) | 20 / 24 | 气动肌肉 / 电动腱绳两版 | 有（指端力） | 气动版 ~\$150K / 电动版 ~\$100K | ❌ |

### 选型要点

```
Allegro: 最常用的"工业级"研究手，16 DoF 每指 4 关节独立，API 成熟
LEAP:    最低成本进入，16 DoF 开源，研究者复现门槛最低
Shadow:  拟人度最高（20 主动 DoF，含腕部），贵且脆弱
Ability: 前臂截肢假肢转研究用，机械紧凑
```

- 研究发表首选 Allegro（兼容性好）与 LEAP（可复现性强）。
- Shadow 适合精细操作 benchmark，但维护成本高。
- 2023-2024 年主流 VLA 论文中 **Allegro 占绝对多数**，2025 年起 LEAP 快速增加。

### 量产灵巧手的关键趋势

1. **非降维全驱动**：Gen 2 开始，Tesla / Figure 走向"每关节一个独立电机"，无耦合约束。代价是电机数量翻倍。
2. **触觉集成**：PaXini DexH13 把指尖、指腹、指背的阵列触觉做成标配；Figure v2 集成指端 force 传感。
3. **成本下探**：2024 年国产量产手单价 ~¥5 万，2025 年因时 RH56DFX ~¥3 万，2026 年 ¥2 万以内是明确目标。
4. **标定复杂度**：腱绳磨损、电机批次差异 → 每只手出厂需独立标定，软件 offsets 表跟随序列号。

## 三、驱动方式对比

| 驱动方式 | 代表 | 优势 | 劣势 | 典型场景 |
|---------|------|-----|-----|---------|
| **电机直驱**（每关节 BLDC/舵机） | Allegro, LEAP, Figure v2 | 响应快、控制精度高、SDK 简单 | 关节体积大、重量集中在手上 | 研究、大 DoF 手 |
| **腱绳驱动** | Shadow（电动版）, Optimus Hand, Ability | 末端轻（电机在前臂）、力重比高 | 磨损、维护、标定复杂；反驱能力差 | 仿生手、人形量产 |
| **气动肌肉**（McKibben） | Shadow（气动版） | 柔顺、安全 | 带气源、控制带宽低 | 研究、柔顺操作 |
| **液压驱动** | Sanctuary Phoenix Hand | 功率密度极高 | 泄漏、维护、噪声 | 高精度/大力抓取 |

### 典型对比

```
研究型 (Allegro / LEAP): 直驱电机 → 16 DoF, 300g-500g, 便于建模
量产型 (Optimus / Figure): 腱绳 + 指端传感 → 16-22 DoF, 500g-800g, 拟人外观
极端精度 (Shadow / Phoenix): 气动/液压 → 20+ DoF, 高成本, 研究 POC
```

**2026 工程共识**：
- 消费级人形 → 腱绳 + 部分直驱混合（末端轻）
- 工业级人形 → 直驱 + 腱绳混合（鲁棒性优先）
- 触觉从"指端一个"转向"阵列覆盖"

## 四、触觉传感器

### 主流技术路线

| 方案 | 原理 | 代表 | 特点 |
|-----|------|------|------|
| **视触觉（Vision-based Tactile）** | 柔性凝胶 + 内部摄像头捕捉形变 | GelSight Mini, Digit, Digit 360 (Meta) | 高分辨率（RGB+深度）；厚度 mm 级；刷新 30-90 Hz |
| **多模态电极阵列** | 电极/压电/温度多通道 | BioTac (SynTouch) | 压力 + 滑动 + 温度 + 振动；拟人指腹；标定复杂 |
| **电容/压阻阵列** | 柔性 PCB 上的容阻阵列 | Contactile, Touchence (JP), 汉威科技, PaXini 内置 | 薄、可大面积覆盖、便宜 |
| **光纤/光学** | 光强变化感知形变 | Luminous, 研究原型 | 抗电磁、柔软 |

### 代表产品参数（2024-2026）

```
GelSight Mini (MIT/GelSight Inc.)
  分辨率: 320×240 (RGB 视触觉)
  厚度: ~15 mm
  刷新: 30 Hz
  单价: ~\$400（学术） / ~\$1500（商用）
  用途: 指端贴装，精细操作研究

Digit (Meta FAIR, 2020)
  规格: 240×320 RGB，12mm 厚
  单价: ~\$300（开源硬件）
  生态: PyTouch / Tac2Pose

Digit 360 (Meta, 2024)
  升级: 球形 360° 覆盖 + 高分辨率触觉
  用途: 灵巧手指尖
  开源: 设计 + 数据集部分开放

BioTac (SynTouch, 2010s-, 现为 Shadow Robot 收购)
  多通道: 19+ 电极 + 压阻 + 温度
  特点: 最接近人类指腹；被 Shadow Hand 广泛使用
  单价: ~\$5-10K/指（昂贵）

PaXini DexH13 内置触觉
  类型: 阵列压阻/电容，指腹 + 指背
  特点: 指手一体设计，整手覆盖

Contactile PapillArray
  澳大利亚公司，9 点阵列，主打摩擦/滑动检测
```

### 触觉集成进 VLA 的早期工作

2024-2025 年触觉仍是 VLA 研究的**边缘通道**，典型路线：

1. **直接特征拼接**：触觉编码成 embedding，与视觉 token 拼接输入 VLM。代表：早期 MIT / CMU 论文、Meta 的 T-DEX。
2. **触觉专用 Encoder + 晚期融合**：GelSight 图像经 CNN/ViT 编码，在 Action Expert 层融合。代表：部分 2025 Diffusion Policy 变体。
3. **触觉预训练**：用 ImageBind / CLIP 思路做视-触对齐预训练，再迁移到 VLA。代表：Meta Sparsh（2024）、部分 Cross-modal 论文。
4. **闭环触觉反馈**：触觉作为高频输入（100-500Hz）进 System 1（类似 Helix），VLM 不直接参与。代表：2025 年 Figure、Optimus 内部方向。

**2026 判断**：触觉 + 视觉的多模态 VLA 仍处早期，**大规模触觉预训练数据集缺失**是主要瓶颈（vs 视觉/语言有 LAION/WebVid 级数据）。AgiBot / 智元等国内厂商开始建设触觉数据采集 pipeline。

## 五、工程难点

### 5.1 非降维全驱动的代价

- **电机数量翻倍**：22 主动自由度 → 22 个微型电机 + 22 套驱动电路。
- **散热**：微型电机高密度排布，连续工作 30 分钟后温升 20-40°C，可能触发降额。
- **标定复杂度**：每只手独立标定各关节零点与刚度，出厂测试环节占制造成本 10-15%。

### 5.2 背部集成（前臂走线）

腱绳方案把电机放到前臂甚至大臂，优势是末端轻，但代价是：

- 腱绳导管磨损，寿命受限（典型 10 万-100 万次循环）
- 多腱交叉耦合，控制需解耦矩阵
- 手腕自由度与腱绳路径冲突

### 5.3 触觉融合的瓶颈

```
带宽: 阵列触觉 ~1000 个通道 × 100Hz = 100k samples/s
     VLM 只能吃 10Hz 级输入 → 必须压缩/摘要
延迟: 触觉物理延迟 <1ms，但软件栈往往 10-50ms
鲁棒: 柔性材料老化、脏污、磨损 → 漂移显著
```

**工程对策**：
- **双频架构**：触觉高频直接入 System 1（反射级），摘要统计入 System 2。
- **Rolling calibration**：运行时持续估计基线漂移。
- **异常检测兜底**：触觉突变 → 紧急停止，而不是完全依赖触觉做闭环。

### 5.4 手掌与手指机构

- **刚柔复合**：指腹用柔性 TPU/硅胶（易抓不规则物体），骨架用铝合金/碳纤维。
- **掌心曲面**：拟人手掌不是平的，需 3D 曲面设计利于包裹。
- **指腹贴合**：触觉传感器上方的覆皮厚度（0.5-2mm）直接影响感度。

### 5.5 寿命问题

| 失效模式 | 典型寿命 | 对策 |
|---------|---------|------|
| 腱绳断裂 | 10 万 - 100 万次 | 更换周期性预防 |
| 电机过热 | 连续 30-60 分钟限功率 | 降额 + 散热设计 |
| 触觉老化 | 3-12 个月漂移显著 | 定期重标定 / 可更换模块 |
| 关节间隙 | 累积 50 万-500 万循环 | 减速器升级 / 更换 |

## 六、2026 趋势与未解问题

### 趋势

1. **触觉进入量产手标配**：PaXini / Figure v2 / Optimus Gen 3 都在集成阵列触觉。
2. **非降维全驱动成为高端标配**：每关节独立电机 + 末端力矩感知。
3. **手-VLA 协同设计**：硬件规格倒推数据格式、时序、带宽。
4. **开源生态扩张**：LEAP Hand、Digit 360 等开源方案推动学术复现。

### 未解

1. **触觉 foundation model**：视觉有 CLIP / DINO，触觉还没有。
2. **视-触对齐预训练**：跨模态标签如何自动获取？
3. **长期可靠性**：每 6-12 个月就需维护的灵巧手无法支撑家庭 C 端。
4. **标定成本**：人工标定每手 30 分钟 - 2 小时，影响规模量产。
