# 执行器与 BOM 成本（2026）

> 人形机器人成本结构中，执行器（关节电机 + 减速器）占 BOM 近半。本文梳理执行器技术路线、选型，拆解双足人形 \$30K 级 BOM，并预测成本拐点。

## 一、执行器技术路线

### 1.1 四条主流路线

```
路线 A: BLDC + 谐波减速器
  特点: 高精度、高刚度、零回差、反驱性差（需力矩传感器）
  单关节成本: \$500-2000
  代表: Franka Panda, Figure 02/03, Optimus 上肢, Apollo
  典型场景: 手臂 / 躯干 / 精细操作

路线 B: 伺服电机 + 行星减速器
  特点: 成本低、刚度高、国产化成熟
  单关节成本: \$200-600
  代表: 国产工业臂 / 部分中国人形 / 优必选 Walker S2
  典型场景: 大扭矩关节 / 成本优先

路线 C: 准直驱 QDD（低减速比 BLDC）
  特点: 反驱极好、力控精度高、适合动态
  单关节成本: \$300-800
  代表: MIT Cheetah → Unitree 四足 / G1/H1 腿部 / Digit 腿部
  典型场景: 腿部（locomotion, 需瞬时大功率 + 反驱）

路线 D: 腱绳驱动
  特点: 末端轻、电机远程布置、机械复杂
  单关节成本: 高（系统级）
  代表: Shadow Hand, Optimus Hand Gen 2 腱绳部分
  典型场景: 灵巧手、仿生
```

### 1.2 路线对比总表

| 维度 | 谐波 | 行星 | 准直驱 QDD | 腱绳 |
|------|------|------|----------|------|
| **减速比** | 50-160:1 | 3-100:1（多级） | 6-10:1 | 视滑轮而定 |
| **回差** | ~0（零回差） | 0.1-0.3° | 无（直驱级） | 腱拉伸带弹性 |
| **力矩密度** | 高 | 中 | 低-中 | 取决于电机 |
| **反驱能力** | 差（自锁倾向） | 中 | 优（透明） | 优 |
| **力控精度** | 需力矩传感器 | 需力矩传感器 | 电流估算即可 | 难（腱非线性） |
| **成本** | 贵（减速器本身贵） | 便宜 | 中（电机大） | 系统级贵 |
| **噪声** | 较低 | 中（齿轮啸叫） | 低 | 低 |
| **寿命** | 长（数千小时） | 中 | 长 | 腱损耗 |

### 1.3 典型选型

```
机器人部位          | 主流路线               | 代表型号
─────────────────────────────────────────────────────
手臂关节（7 DoF）     | BLDC + 谐波           | Franka / Figure v2
躯干                 | BLDC + 谐波（小）     | Optimus
髋/膝/踝（腿部）      | 准直驱 QDD 或 大行星   | Unitree H1 / Digit
颈部                 | 小型伺服 + 行星/谐波   | —
手指                 | 微型 BLDC + 腱绳      | Optimus Hand
末端关节（腕）        | 谐波（精度优先）       | 各家通用
```

**2026 趋势**：
- 腿部全面转向 **QDD**（动态 locomotion 需求）
- 手臂继续以**谐波**为主，但中国厂商开始用**高精度行星**（绿的谐波 + 纽氏减速）替代降本
- 灵巧手 22 DoF 级别开始直驱化（电机微型化进步）

## 二、关键零部件供应链

### 2.1 减速器

| 类型 | 主要玩家 | 国产化程度 | 单价（工业级） |
|-----|---------|----------|--------------|
| **谐波减速器** | Harmonic Drive（日，市场份额 ~50%）、绿的谐波（中）、Leaderdrive 来福谐波、Spinea（斯洛伐克） | 较好（绿的/来福已配套多家人形） | \$200-800 |
| **行星减速器** | Neugart（德）、Apex（台）、纽氏传动（中）、中大 | 好 | \$50-300 |
| **RV 减速器** | Nabtesco 纳博特斯克（日，市场份额 ~60%）、双环、秦川、中大力德 | 中等（高端仍依赖日本） | \$500-2000 |

- **谐波** vs **RV**：谐波主要用于手臂/腕部（精度高），RV 用于底座/大扭矩（工业臂第一关节）。人形机器人以谐波为主。
- **绿的谐波**（苏州绿的）：国内唯一谐波上市公司，配套 Unitree / 优必选 / 傅利叶 / 智元等。
- **Harmonic Drive**（日本，含 Samick、Kaydon 系）：全球高端标杆，但交期长、成本高。

### 2.2 电机

| 类型 | 主要玩家 | 备注 |
|-----|---------|------|
| **工业 BLDC** | Maxon（瑞士，高精度）、Parker、Kollmorgen | 研究平台首选 |
| **机器人用 BLDC** | Moog、ElmoMC | 高端人形 |
| **国产机器人电机** | 雷赛智能、鸣志电器、汇川、禾川、步科 | 价格优势，2024-2026 快速替代 |
| **超紧凑 BLDC**（灵巧手） | Faulhaber（德）、Maxon EC-series | 微型高功率密度 |
| **QDD 电机** | T-Motor（中，Unitree/Cheetah 系）、CubeMars | 机器狗/腿部主力 |

### 2.3 高精度编码器

- **绝对式编码器**：RLS（英）、Renishaw（英）、海德汉（德）— 高端主导
- **磁编码器**：AMS（奥地利）、霍尔森（中）— 集成化

**卡脖子**：高端绝对式光学编码器仍由欧洲主导，国产替代在磁编码器上较快。

### 2.4 力/力矩传感器

- **六维力矩传感器**：ATI（美）、坤维科技（中）、宇立仪器（中）— 手腕标配
- **单轴力矩传感器**：集成在谐波减速器输出端，关节力控依赖

### 2.5 一体化关节模组（厂商 + 供应）

2024-2026 年出现一个新类别：**一体化关节**（电机 + 减速器 + 编码器 + 驱动器集成）。

- 国际：Harmonic Drive FHA 系列、Kollmorgen KBM
- 国内：禾川、雷赛、星动/傅利叶自研 FSA 系列、本末 QDD 关节
- 优势：人形机器人厂商直接采购，降低组装复杂度
- 单价：\$500-2000（视规格）

## 三、BOM 拆解：双足人形 \$30K 级成本构成

### 3.1 中端双足人形整机 BOM（2025-2026 估算）

以 Figure 02 / Unitree G1 高配 / Apollo 级（\$30-40K BOM）为典型：

```
总成本 \$30-40K 分布:
├── 执行器（28 关节 + 减速器 + 编码器）   40-45%  \$12-18K
│   ├── 谐波减速器: 12-16 个 × \$300-600     = \$4-9K
│   ├── 行星/其他减速器: 10-14 个 × \$80-200  = \$1-3K
│   ├── 电机: 28 × \$100-250                = \$3-7K
│   └── 编码器 + 力矩传感器: 28 × \$50-150   = \$1.4-4K
├── 灵巧手 × 2                          10-15%  \$3-6K
│   └── 量产手成本 \$1.5-3K 每只
├── 传感器（IMU, RGB-D, 触觉）            5-10%  \$1.5-4K
│   ├── RGB-D × 3-4: \$300-800
│   ├── IMU × 2: \$100-300
│   ├── 触觉（指端+腕）: \$500-2000
│   └── 关节力矩传感器: 已算入执行器
├── 计算平台（Jetson Thor / Orin）        5-8%   \$1.5-3K
├── 结构件（碳纤维 + 铝合金 + 3D 打印）    10-15%  \$3-5K
├── 电池 + BMS                          5-8%   \$1.5-3K
│   └── ~1 kWh，含 BMS
├── 电源管理 / 线束 / 外壳               8-10%  \$2.5-4K
└── 总装 / 测试 / 校准                   5-10%  \$1.5-4K
```

**关键观察**：
- **执行器占总成本 40-45%**，是降本主战场
- 灵巧手 + 触觉合计 10-15%，且在高端型号中占比持续上升
- 电池占比相对小（~5-8%），但续航仍是体验瓶颈
- 总装/测试占 5-10%，标定环节成本不可忽视

### 3.2 中国视角的人民币 BOM（2025）

```
¥200K 国产人形机器人 BOM 估算:
├── 关节执行器总成 (28 × ¥3-5K)      ¥100-140K
│   └── 国产谐波 ¥600-1500/个, 国产电机 ¥400-1000/个
├── 灵巧手 × 2 (国产因时级)           ¥10-20K
├── 主控 + 感知模组                   ¥15-25K
│   └── Jetson Orin ¥8-15K, 相机/触觉约 ¥5-10K
├── 电池 + 电源管理                   ¥5-10K
├── 结构件 + 外壳                     ¥10-15K
├── 传感器（IMU, 视觉, 触觉）         ¥5-10K
└── 其他（线束、紧固件、总装、测试）   ¥15-20K
```

**对比**：国产中端与美系中端同规格差距约 1.5-2x（¥200K vs \$30-40K ≈ ¥210-280K），差距主要来自 Jetson 芯片、部分高端减速器、编码器进口。

## 四、成本拐点预测

### 4.1 历史曲线

```
2013-2020: Atlas 液压老款  ~\$2M (估算, 未对外售)
2022:     Atlas 电动 / Apollo 原型      ~\$200-400K
2023:     Digit v3 / Unitree H1 早期    ~\$100-150K
2024:     Figure 02 / Apollo 量产       ~\$50-100K
         Unitree G1（低配）起步         ~\$14-16K
2025:     头部双足人形 BOM            ~\$30-50K
         国产中端整机                   ¥150-250K
2026 (预期): 批量订单 BOM             ~\$20-35K
         Optimus Gen 3 成本目标       ~\$20-30K
2028 (预期): 家用起步价               ~\$10-20K
2030 (远期): 大众市场                  <\$10K
```

### 4.2 降本驱动力

1. **规模化生产**：从年产几百台 → 年产万台，单位固定成本快速摊薄（以 Tesla Gigafactory 式做法降本 30-50%）。
2. **中国供应链替代**：
   - 谐波减速器：绿的谐波 vs Harmonic Drive，价格约 50-70%
   - 电机：汇川/禾川 vs Maxon，价格约 40-60%
   - RGB-D 相机：Orbbec vs Intel RealSense，价格约 50-70%
3. **结构件减重与塑化**：碳纤维 + 工程塑料替代铝合金，重量降 20-30%、成本降 15-25%。
4. **标定自动化**：机器视觉辅助自动标定，降低人工占比。
5. **一体化关节模组**：从"电机 + 减速器 + 驱动器"分别采购到整体模组，降本 15-20%。

### 4.3 降本的不确定性

```
乐观路径（Tesla/Figure 执行到位）:
  2026 年 Q4 头部厂商 BOM 达 \$20-25K
  2027 年对外售价进入 \$30-50K 区间
  工业 B 端 ROI 降至 18-30 月

悲观路径（需求不及预期）:
  产能爬坡不到位 → 规模效应不足
  高端芯片/减速器缺货 → 成本停滞
  2026 年仍在 \$35-50K BOM
```

**判断**：头部两三家（Tesla、Figure、Unitree 国内）2026 年能下探到 \$20-35K；腰部/尾部仍停留在 \$40-80K。

## 五、BOM 结构的长期演变

```
2024:  执行器 45% + 电子 15% + 结构/外壳 15% + 其他
2026:  执行器 40% + 电子 20% + 结构/外壳 12% + 灵巧手 12% + 电池 8% + 其他
2028:  执行器 35% + 电子 22% + 灵巧手 15% + 电池 10% + 结构 10% + 其他
2030:  执行器 30% + 电子 25% + 灵巧手 15% + 电池 12% + 其他
```

**结构变化的含义**：
- 执行器占比下降（规模化成熟得最早）
- **灵巧手占比上升**（非降维 + 触觉集成 + 更复杂）
- 电子（计算 + 传感）占比上升（VLA 需更强端侧芯片）
- 电池占比上升（续航改善对应更大电量）
