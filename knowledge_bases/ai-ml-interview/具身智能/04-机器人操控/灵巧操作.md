# 灵巧操作 (Dexterous Manipulation)

灵巧操作指用多指灵巧手完成人类手才能做的精细任务：手内旋转、工具使用、拧瓶盖、穿针引线、折纸等。2025–2026 年，这一方向从纯研究走向量产手落地，成为具身智能最活跃的战场。

## 1. 灵巧手硬件全景 (2026)

| **Allegro Hand** | Wonik Robotics | 4 | 16 | 电机直驱 | 研究默认选择 |
| **LEAP Hand** | CMU | 4 | 16 | 电机直驱 | 开源低成本（约 2000 美元） |
| **Ability Hand** | Psyonic | 5 | 6–10 | 电机 + 欠驱动腱 | 商业假肢起源 |

### 量产方向的共识与分歧

- **共识**：5 指拟人形态、指尖触觉必备、混合驱动（腱 + 直驱）降成本。
- **分歧**：自由度到底该多少？Tesla Gen2 用 22 DoF 极致拟人，Figure 和 1X 选择简化到 16 DoF 以提升可靠性。

## 2. 经典任务

| 任务 | 描述 | 代表工作 |
|------|------|---------|
| **In-hand Rotation** | 手内重新定向物体（魔方、立方体） | OpenAI Rubik's Cube 2019, Hora, In-Hand Rotation via Visual Feedback |
| **Pen Spinning** | 笔旋转，对精细时序控制要求极高 | Eureka (NVIDIA 2023), DrEureka 2024 |
| **Tool Use** | 使用锤子、螺丝刀、剪刀 | DexArt, RobotLearnTools |
| **Object Reorientation** | 把物体调到目标姿态 | DexPoint, Visual Dexterity |
| **Manipulation with Vibration / Screws** | 拧螺丝、敲钉子 | DexForce, Tactile Manipulation |
| **Piano / Keyboard** | 多指协调按键 | RoboPianist (Google 2023) |
| **Rope / Cable** | 绕线、打结 | DeformableManipulation benchmarks |

## 3. 方法演进

### 3.1 基于 RL 的灵巧操作

#### OpenAI Rubik's Cube (2019) — 里程碑

- 在 MuJoCo 仿真中训练 PPO
- **大规模域随机化**（ADR, Automatic Domain Randomization）：摩擦、质量、尺寸、光照自动调整难度
- Sim-to-Real 零样本迁移到真实 Shadow Hand
- 证明：**仿真 + 域随机化 + 大算力 = 灵巧操作可行**
- 但：训练耗时数月、电力成本百万美金、仅一个任务

#### 2023–2025 RL 路线发展

| 工作 | 年份 | 贡献 |
|------|------|------|
| **Hora** (MIT) | 2023 | In-hand rotation 泛化到多种物体 |
| **Visual Dexterity** | 2023 | RGB + 本体感觉混合输入 |
| **Eureka** (NVIDIA) | 2023 | GPT-4 自动生成奖励函数，Pen Spinning 达人类水平 |
| **DrEureka** | 2024 | 自动生成奖励 + 域随机化 |
| **DexPoint** (Wang et al.) | 2023 | 点云输入的 RL 抓取，泛化到 3000+ 物体 |
| **DexGraspNet** | 2023 | 大规模灵巧抓取数据集 (132 万抓取) |
| **DexH2R** (UCSD) | 2024 | Human-to-Robot 灵巧动作迁移 |
| **ViTaL / AnyRotate** | 2025 | 触觉 + 视觉融合的 in-hand rotation |

### 3.2 遥操作 + 模仿学习

RL 需要仿真，模仿学习则用人类专家数据直接教机器人。

| 系统 | 年份 | 捕捉方式 | 亮点 |
|------|------|---------|------|
| **AnyTeleop** (UCSD) | 2023 | RGB 视觉手部追踪 | 统一遥操作框架 |
| **DexPilot** (NVIDIA) | 2020 | RGB-D 手部追踪 | 早期遥操作系统 |
| **Bunny-VisionPro** | 2024 | Apple Vision Pro | 用户友好的双手遥操作 |
| **Open-TeleVision** (MIT) | 2024 | Apple Vision Pro + 立体视觉 | 沉浸式远程灵巧手遥操作 |
| **UMI** (Universal Manipulation Interface, Stanford) | 2024 | 手持夹爪 + 鱼眼相机 | 数据采集不依赖机器人，跨具身迁移 |
| **Dex-UMI / DexUMI** | 2025 | 穿戴式手套 + 视觉 | UMI 扩展到灵巧手 |
| **ALOHA 2** | 2024 | 双臂主从遥操作 | 成熟商业化开源硬件 |

#### Open-TeleVision (MIT 2024) 关键贡献

1. 通过 Vision Pro 的立体显示让操作者获得机器人的**第一人称立体视角**。
2. 手部和头部位姿通过 Vision Pro SDK 实时流式传输。
3. 显著降低遥操作的 embodiment gap，使操作者"成为"机器人。
4. 开源 + 低成本（相对 VR 手套方案）。

### 3.3 从人类视频学习

- **Video Pretraining**：用 Ego4D、SomethingSomething V2、HOI4D 预训练手-物交互表征。
- **H2O / HO-3D**：人手-物体交互数据集。
- **MimicPlay / Vid2Robot**：视频演示 → 机器人动作的跨域迁移。
- 2026 趋势：视频预训练 + 少量真机 fine-tune，是数据最经济的路线。

### 3.4 VLA 时代的灵巧操作 (2025-2026)

- **π₀ / π₀.5** 原生支持灵巧手动作空间（16+ DoF），dual-system 架构：VLM 出意图 + 高频 action expert 出动作。
- **GR00T N1 / N2** (NVIDIA) 针对人形 + 灵巧手预训练。
- **Helix** (Figure) 单一 35-DOF 网络输出全身 + 双手动作。
- **RDT-2** (清华+智元) MoE 动作头，双臂灵巧手统一建模。

## 4. 2026 前沿方向

### 4.1 触觉融合

视觉 + 本体感觉无法捕捉接触是否发生、是否打滑、用了多少力。触觉成为灵巧操作的"第三只眼"。

| 传感器 | 类型 | 代表工作 |
|--------|------|---------|
| **GelSight** / **GelSight Mini** | 视觉触觉（凝胶+相机） | GelSight-FM, Touch & Go |
| **Digit** / **Digit 360** (Meta) | 视觉触觉 | T3, AnyTouch |
| **BioTac** (SynTouch) | 压阻 + 振动 | 早期工业触觉 |
| **PaXini 触觉** | 柔性电子皮肤 | 国产集成方案 |
| **Tesla 指尖触觉** | 电容/电阻阵列 | Optimus Hand Gen2 |

**VLA + 触觉** 的典型模式：
```
视觉 token + 语言 token + 触觉 token → Transformer → 动作
```
触觉 token 化方案：CNN/ViT 编码触觉图像，或直接 patch + positional encoding。

### 4.2 从研究到量产

2026 年灵巧手量产的三道坎：

1. **成本**：研究级 Shadow Hand ≈ 10 万美元；量产目标 < 5000 美元（Tesla/Figure 宣称 3000–5000）。
2. **可靠性**：研究任务成功率 60–80% 即可发论文；量产要 > 99%，否则无法进生产线。
3. **数据**：VLA 需要数千小时遥操作数据，这是量产公司内部最大资产。

### 4.3 开放问题

- **形变物体灵巧操作**：布料折叠、电缆、食物，仍是开放挑战。
- **长时序灵巧任务**：组装、烹饪等需要分钟级执行，VLA 的 context 限制仍是瓶颈。
- **双手协同**：当前多数工作是单手，双手操作（剪纸、系鞋带）数据极少。
- **精度 / 力控制**：螺丝装配这类高精度任务需要亚毫米级 + 精准力控，纯 VLA 很难达到。
