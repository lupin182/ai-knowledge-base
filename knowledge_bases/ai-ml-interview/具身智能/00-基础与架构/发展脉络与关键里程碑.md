# 发展脉络与关键里程碑

> 本文梳理具身智能从 1960s 工业机器人时代到 2026 年大模型驱动时代的关键里程碑，重点展开 2022-2026 年 VLA 与世界模型的核心事件。

## 一、三段式宏观脉络

```
传统机器人 (1960s-2000s)     学习驱动 (2010s-2022)        大模型驱动 (2023-)
─────────────────────────────────────────────────────────────────────
手工编程/PID控制     →    Deep RL / IL           →   VLM + VLA
结构化环境           →    仿真训练               →   Foundation Model
单任务 / 单本体      →    多任务泛化             →   通用具身智能 + 端侧部署
```

---

## 二、阶段一：传统机器人（1960s-2000s）

经典控制论与规划时代，智能高度依赖手工建模与结构化环境。

| 年份 | 事件 |
|------|------|
| **1961** | Unimate 机器人在 GM 工厂部署，工业机器人时代开启 |
| **1968** | SRI Shakey 机器人，首个结合感知-规划-执行的 AI 机器人 |
| **1980s** | PUMA 560 出现，机械臂研究进入大学实验室 |
| **1986** | Rodney Brooks 提出 Subsumption Architecture，反对纯符号 AI，主张具身 |
| **1990s** | RRT 运动规划算法（LaValle）、Behavior-Based Robotics |
| **1997** | RoboCup 首届机器人足球赛 |
| **2001** | Boston Dynamics BigDog 原型，开启现代足式研究 |

核心范式：**手工建模 + PID/阻抗 + 符号规划 + 结构化环境**。

---

## 三、阶段二：学习驱动（2010s-2022）

深度学习 + 大规模仿真 + 强化学习/模仿学习推动研究。

| 年份 | 事件 | 意义 |
|------|------|------|
| **2012** | AlexNet；深度学习进入视觉主流 | 感知层革命 |
| **2015** | DQN 玩 Atari（DeepMind） | 深度 RL 诞生 |
| **2016** | OpenAI Gym 发布 | 统一 RL 接口 |
| **2016** | Levine et al. "End-to-End Training of Deep Visuomotor Policies" | 真正端到端的视觉-运动策略 |
| **2017** | Domain Randomization（OpenAI） | Sim2Real 关键技术 |
| **2018** | Habitat / iGibson 发布 | 大规模视觉导航仿真 |
| **2018** | OpenAI Dactyl 单手魔方 | RL + Domain Randomization 里程碑 |
| **2019** | Boston Dynamics Spot 商业化 | 足式机器人量产 |
| **2020** | DexMV / DexYCB benchmark | 灵巧操作数据化 |
| **2021** | RMA（Rapid Motor Adaptation） | 四足稳健行走代表 |
| **2022.04** | SayCan（Google） | LLM 作为机器人规划器的开山之作 |
| **2022.07** | VIMA / Perceiver-Actor | 多模态 prompt 驱动策略 |
| **2022.12** | Diffusion Policy（Chi et al.） | 扩散模型进入操控 |
| **2022.12** | RT-1（Google） | 13 万条真机数据的 Transformer Policy |
| **2022.12** | ChatGPT 发布 | 间接引爆 2023 具身大模型浪潮 |

核心范式：**CNN/Transformer + RL/IL + 大规模仿真**。

---

## 四、阶段三：大模型驱动（2023-至今）

### 4.1 2023：VLA 范式诞生

| 日期 | 事件 |
|------|------|
| **2023.03** | PaLM-E（Google） — 多模态具身语言模型 |
| **2023.07** | RT-2 — **第一个 VLA**，将动作表示为文本 token |
| **2023.10** | Open X-Embodiment 发布，22 种机器人、80 万轨迹共享 |
| **2023.12** | Mobile ALOHA / ALOHA 2，低成本双臂遥操作平台 |

**范式意义**：VLA 证明了"用预训练 VLM + 机器人数据微调"是可行路径，引爆后续两年。

### 4.2 2024：扩散与 Flow Matching 动作头成熟

| 日期 | 事件 |
|------|------|
| **2024.05** | Octo（Berkeley） — 开源跨本体 Transformer + Diffusion 动作头 |
| **2024.06** | OpenVLA（Stanford） — 7B 开源 VLA，Llama 2 + 离散 token |
| **2024.10** | **π₀**（Physical Intelligence） — 首个 Flow Matching VLA，10K 小时数据 |
| **2024.10** | OpenVLA-OFT — 并行解码把控制频率从 26Hz 拉到 109Hz |
| **2024.11** | CogACT（MSR Asia） — 解耦认知与动作，30Hz SOTA |
| **2024.11** | RDT-1B（清华） — 双臂扩散基础模型 |
| **2024.02** | Genie 1（DeepMind） — 从无标签视频学可交互世界模型 |
| **2024.12** | Genie 2（DeepMind） — 可控 3D 场景世界模型 |

### 4.3 2025：双系统架构成为行业共识

| 日期 | 事件 |
|------|------|
| **2025.01** | **NVIDIA Cosmos** — CES 发布"世界基础模型"平台，Apache 2.0 开源 |
| **2025.02** | **Figure Helix** — 首个端到端 VLA，7B VLM (S2, 9Hz) + 80M Transformer (S1, 200Hz)，部署 Figure 02 |
| **2025.03** | **NVIDIA GR00T N1** — 首个 Apache 2.0 开源 VLA，面向人形，Eagle-2 VLM + DiT |
| **2025.03** | **Gemini Robotics + Gemini Robotics-ER**（DeepMind）— Gemini 2 驱动，后者专注具身推理 |
| **2025.03** | DeepMind 与 Apptronik 深度合作，Gemini Robotics 部署 Apollo |
| **2025.04** | **π₀.5**（PI） — 家庭开放世界泛化，加入高层语言推理 |
| **2025.05** | SmolVLA（HF LeRobot） — ~500M 端侧可跑 VLA |
| **2025.06** | Meta V-JEPA 2 — 非生成式隐空间世界模型，zero-shot planning |
| **2025.08** | Genie 3（DeepMind，据披露） — 更长一致性与物理约束 |
| **2025 H2** | **π-fast** — PI 的低延迟轻量版，目标 30Hz+ 闭环 |
| **2025 H2** | **Helix v2** — 部署 Figure 03，更长 horizon 推理 |
| **2025 H2** | **GR00T N1.5 / N2** — 更大动作专家、触觉输入、与 Cosmos 联合训练 |
| **2025** | 人形机器人量产爆发：Digit 在 GXO 部署百台级、Figure 在 BMW 部署、Unitree G1 量产 |
| **2025** | 智元 GO-1 / UniAct、银河通用 Galbot、自变量机器人等国内 VLA 密集发布 |

### 4.4 2026：多模态推理、端侧部署、世界模型耦合

| 日期（披露 / 预期） | 事件 |
|---------------------|------|
| **2026 H1（据披露）** | **π₀.6 / π₀.7**（PI）— 迭代升级，重点多模态推理、长 horizon、家庭泛化；π₀.7 预期融合更强语言规划与世界模型反馈 |
| **2026（预期）** | **Cosmos 2.0**（NVIDIA） — 更强视频世界模型，深度整合 GR00T 数据飞轮 |
| **2026（预期）** | Jetson Thor（2000 TOPS）规模化出货，7B 级 VLA 端侧部署常态化 |
| **2026（预期）** | RDT-2（清华/星动）正式开源，MoE 动作头 |
| **2026（预期）** | 数据飞轮闭环形成：真机 → 世界模型增广 → VLA 训练 → 更好策略 → 更多真机数据 |

> 2025 H2 之后的条目带"据披露/预期"标识，具体日期与能力以官方发布为准。

## 六、产业并行事件（不可忽视）

```
学术里程碑                    产业里程碑
─────────────────────────────────────────────────────
RT-2 (2023.07)              Figure 成立 (2022) → 02 量产
π₀ (2024.10)                Physical Intelligence 成立 (2024.03)
GR00T N1 (2025.03)          NVIDIA Isaac / Thor 产品化
Helix (2025.02)             Figure 02 在 BMW 工厂部署
Cosmos (2025.01)            Agility Digit 在 GXO 百台部署
π₀.5 (2025.04)              Unitree G1 量产 (~10 万元)
                            1X Neo 家庭订阅 (2025)
                            宇树/智元/银河通用等国内厂商融资扩张
```

产业与学术的"双飞轮"是 2025-2026 年的最大特征。真机部署产生数据，数据反哺模型，模型带来更好部署——这是 2023 年之前从未实现过的闭环。
