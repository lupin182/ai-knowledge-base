# 李群、李代数与四元数

> 具身智能里一切跟「转动 / 位姿」有关的表示、积分、插值与优化，背后都是同一套数学：**李群**（姿态住的弯曲空间）、**李代数**（它的平直切空间，让你能加减、求导、优化）、**四元数**（表示 3D 旋转最紧凑实用的坐标）。本文按课件形式从「为什么需要它」讲到 SO(3)/SE(3)、so(3)/se(3)、exp/log、四元数运算、SLERP、深度学习里的 6D 表示，以及在 SLAM、IMU、Diffusion Policy、人体动作中的落地。
>
> **先修**：线性代数（矩阵、特征值）、一点点微积分。**记号约定**：粗体小写 $\mathbf{v}$ 为向量，大写 $R,T$ 为矩阵，$\hat{(\cdot)}$ 表示「向量 → 反对称矩阵」。

---

## 一、动机：为什么具身智能逃不开这套数学

智能体要在 3D 物理世界里行动——机械臂末端、相机、无人机机身、人体关节、被抓的物体——每个刚体的**位姿（pose）= 位置（translation）+ 朝向（orientation）**。

- **位置**好办：$\mathbf{t}\in\mathbb{R}^3$，是普通向量，可以直接相加、求平均、做梯度下降。
- **朝向**是麻烦的那一半：它住在一个**弯曲的空间**上，不是向量空间。两个旋转「相加」没有意义，朝向之间的「距离」也不是欧氏距离。

一句话点题:

> **平移活在平的 $\mathbb{R}^3$ 里，旋转活在弯的 $SO(3)$ 上。处理弯空间的通用工具，就是李群 + 李代数；四元数则是 $SO(3)$ 这个特定弯空间的一套好用坐标。**

具身智能里这套数学出现在哪些地方:

| 场景 | 用到什么 |
|------|----------|
| IMU / 里程计积分 | 把角速度（so(3) 元素）积分成朝向 → `exp` 回 SO(3) |
| 视觉 SLAM / Bundle Adjustment | 在 SE(3) 流形上联合优化相机位姿 |
| 6DoF 抓取位姿估计 | 预测物体/夹爪在 SE(3) 中的位姿 |
| Diffusion Policy / VLA 动作头 | 末端位姿增量的旋转分量该用什么表示（6D / 四元数） |
| 人体动作 / SMPL | 每个关节是一个 SO(3) 旋转，一段动作 = SO(3) 上的轨迹 |
| 逆运动学 / 运动规划 | 配置空间含 SO(3)/SE(3)，要在流形上搜索、插值 |

掌握它的收益是**统一**：一旦理解了「在李代数里做增量、用 exp 映射回李群」这一个范式，上面所有问题的优化、滤波、插值就是同一套写法。

## 二、热身：旋转到底有几种写法

在进入李群之前，先把工程里常见的几种旋转表示摆出来，建立全局观。它们描述的是**同一个东西**（一个 3D 旋转），只是坐标不同。

| 表示 | 参数量 | 约束 | 优点 | 致命缺点 |
|------|--------|------|------|----------|
| 旋转矩阵 $R$ | 9 | $R^TR=I,\det R{=}1$（6 个约束） | 直接作用于向量、复合简单 | 冗余、数值上会漂移出 $SO(3)$ |
| 欧拉角 (roll-pitch-yaw) | 3 | 无 | 人类可读、好调参 | **万向锁**、不唯一、不连续 |
| 轴角 / 旋转向量 $\boldsymbol\phi=\theta\mathbf{u}$ | 3 | 无（最小参数化） | 最小、就是 so(3)、适合做增量 | $\theta$ 大时有 $2\pi$ 周期奇异 |
| 单位四元数 $q$ | 4 | $\|q\|=1$（1 个约束） | 无万向锁、复合快、易插值、易归一化 | **双重覆盖** $q\equiv-q$ |
| 6D 表示（两列） | 6 | 无（网络输出后正交化） | **连续**，对神经网络友好 | 不是最小、需 Gram-Schmidt |

> **核心矛盾**：旋转的「真实自由度」只有 **3**，但用 3 个数表示一定会在某处出现奇异/不连续（这是拓扑决定的，见 §八）。所以工程上常用「冗余但无奇异」的表示（矩阵 / 四元数 / 6D）来**存储和传播**，用「最小但局部」的表示（轴角 = so(3)）来做**增量和优化**。

下面正式进入主题，先讲李群（旋转住的空间），再讲李代数（做增量的空间），最后讲四元数（最实用的坐标）。

## 三、李群：姿态本身住的空间

### 3.1 直觉定义

> **李群（Lie group）= 既是「群」又是「光滑流形」的集合，且群的乘法和求逆都是光滑（可微）的。**

拆成两半理解:

- **群（Group）**：一个集合 + 一个「乘法」运算，满足
  1. **封闭性**：两个元素相乘还在集合里（两个旋转复合还是旋转）；
  2. **结合律**：$(AB)C = A(BC)$；
  3. **单位元**：存在 $e$ 使 $eA=Ae=A$（「不转」这个旋转）；
  4. **逆元**：每个 $A$ 有 $A^{-1}$（反着转回去）。
- **流形（Manifold）**：一个局部看起来像平直欧氏空间、但整体可以弯曲的空间（球面就是 2 维流形——局部像平面，整体是弯的）。**光滑**意味着没有尖角，处处可求切线/切平面。

把两者合起来：李群是一个「可以做乘法、并且这个空间是光滑弯曲的」对象。具身智能里最重要的两个李群是 $SO(3)$ 和 $SE(3)$。

### 3.2 $SO(3)$：3D 旋转群

$$SO(3) = \{\, R\in\mathbb{R}^{3\times3} \;:\; R^{\top}R = I,\ \det R = +1 \,\}$$

- **名字**：Special（$\det=+1$，排除镜像翻转）Orthogonal（$R^\top R=I$，保长保角）group，维度 3。
- **群运算**：矩阵乘法。$R_1R_2$ 表示「先做 $R_2$ 再做 $R_1$」。
- **单位元**：单位矩阵 $I$（不旋转）。
- **逆元**：$R^{-1}=R^{\top}$（正交矩阵的转置就是逆，反着转）。
- **自由度**：9 个矩阵元素被 6 个正交约束限制，剩 **3 个自由度**——这正是「绕哪个轴、转多少角度」的 3 个数。

**关键认知：$SO(3)$ 是弯的。** 两个旋转矩阵逐元素相加 $R_1+R_2$ 一般**不再是旋转矩阵**（不满足正交约束）。所以「平均两个朝向」「朝向 + 小扰动」都不能用普通加法——这就是后面需要李代数的根本原因。

### 3.3 $SE(3)$：刚体变换群（位姿）

一个完整的刚体位姿 = 旋转 + 平移，用 $4\times4$ 齐次矩阵表示:

$$T = \begin{pmatrix} R & \mathbf{t} \\ \mathbf{0}^\top & 1 \end{pmatrix},\qquad R\in SO(3),\ \mathbf{t}\in\mathbb{R}^3$$

- **名字**：Special Euclidean group，维度 6（旋转 3 + 平移 3）。
- **作用在点上**：把一个 3D 点 $\mathbf{p}$ 写成齐次坐标 $\tilde{\mathbf{p}}=(\mathbf{p},1)^\top$，则 $T\tilde{\mathbf{p}} = (R\mathbf{p}+\mathbf{t},\,1)^\top$——先旋转再平移。
- **群运算**：矩阵乘法，自动处理「两次变换的复合」。
- **逆元**：$T^{-1}=\begin{pmatrix} R^\top & -R^\top\mathbf{t}\\ \mathbf{0}^\top & 1\end{pmatrix}$。

> **为什么用 $4\times4$ 齐次矩阵**：把「旋转 + 平移」这个仿射操作变成一次纯矩阵乘法，复合多个变换就是连乘 $T_1T_2T_3\cdots$，求逆、链式坐标变换都统一了。SLAM 里相机轨迹、机械臂的正运动学（DH 变换连乘）全靠它。

### 3.4 一张图看懂「群 + 流形」

```
        SO(3) 是一个弯曲的 3 维流形（这里画成 2 维球面示意）
                          ┌───────────────┐
              R₁ •────────┤   切平面       │  ← 在某点 R₁ 处的切空间
                  ╲       │  (平直，可加减) │     是一个平直向量空间
                   ╲      └───────────────┘
        曲面上的点 = 一个旋转
        曲面本身弯曲 ⇒ 不能用直线(向量加法)连接两点
        但每一点的"切平面"是平的 ⇒ 可以在切平面里做微积分

        特别地：在单位元 e=I 处的切平面，就是"李代数" so(3)
```

## 四、李代数：在单位元处的「切空间」

### 4.1 直觉：把弯空间在一点处「拉平」

旋转不能直接相加，但「**无穷小**旋转」可以。想象朝向随时间连续变化 $R(t)$，在 $t=0$ 时 $R(0)=I$，那么它的**变化率** $\dot R(0)$ 就是单位元处的一个切向量。所有这些切向量构成的空间，就是李代数。

> **李代数（Lie algebra）= 李群在单位元 $e$ 处的切空间。它是一个平直的向量空间，可以自由地做加减、数乘、求导。**

李代数像李群一样有名字：$SO(3)$ 的李代数记作 $\mathfrak{so}(3)$，$SE(3)$ 的记作 $\mathfrak{se}(3)$（小写哥特体）。

### 4.2 $\mathfrak{so}(3)$：反对称矩阵 ≅ 旋转向量

对正交约束 $R(t)^\top R(t)=I$ 两边求导，在 $t=0$（$R=I$）处:

$$\dot R^\top R + R^\top \dot R = 0 \ \Longrightarrow\ \dot R^\top + \dot R = 0$$

也就是 $\dot R$ 是一个**反对称矩阵**。所以:

$$\mathfrak{so}(3) = \{\, A\in\mathbb{R}^{3\times3} : A^\top = -A \,\}$$

反对称矩阵只有 3 个独立元素，所以 $\mathfrak{so}(3)\cong\mathbb{R}^3$。这个同构通过 **hat 算子 $\wedge$**（向量 → 反对称矩阵）实现:

$$\boldsymbol\phi=\begin{pmatrix}\phi_1\\\phi_2\\\phi_3\end{pmatrix}\ \xrightarrow{\ \wedge\ }\ \boldsymbol\phi^{\wedge}=\begin{pmatrix} 0 & -\phi_3 & \phi_2 \\ \phi_3 & 0 & -\phi_1 \\ -\phi_2 & \phi_1 & 0 \end{pmatrix}$$

反过来用 **vee 算子 $\vee$**（反对称矩阵 → 向量）。这个 $\boldsymbol\phi\in\mathbb{R}^3$ 有清晰的物理意义:

- **作为旋转向量（轴角）**：方向 $\mathbf{u}=\boldsymbol\phi/\|\boldsymbol\phi\|$ 是转轴，模长 $\theta=\|\boldsymbol\phi\|$ 是转角。
- **作为角速度**：若 $\boldsymbol\phi=\boldsymbol\omega\,\Delta t$，则 $\boldsymbol\omega$ 就是角速度向量。

> **一个有用的恒等式**：反对称矩阵乘向量 = 叉乘。$\ \boldsymbol\phi^{\wedge}\mathbf{v} = \boldsymbol\phi\times\mathbf{v}$。这把「李代数作用」和熟悉的叉积联系起来了。

### 4.3 指数映射：李代数 → 李群（Rodrigues 公式）

李代数是「无穷小旋转」，要变成「有限旋转」，靠**指数映射 $\exp$**（就是矩阵指数 $\exp(A)=\sum_k A^k/k!$）。对 $\mathfrak{so}(3)$，这个级数有闭式解，即著名的 **Rodrigues 旋转公式**。设 $\boldsymbol\phi=\theta\mathbf{u}$（$\mathbf{u}$ 单位轴，$\theta$ 转角）:

$$\boxed{\,R = \exp(\boldsymbol\phi^{\wedge}) = I + \frac{\sin\theta}{\theta}\,\boldsymbol\phi^{\wedge} + \frac{1-\cos\theta}{\theta^{2}}\,(\boldsymbol\phi^{\wedge})^{2}\,}$$

或用单位轴 $\mathbf{u}$ 写成更常见的形式:

$$R = I + \sin\theta\,\mathbf{u}^{\wedge} + (1-\cos\theta)(\mathbf{u}^{\wedge})^{2}$$

**直觉**：$\exp$ 把「在切平面上沿方向 $\boldsymbol\phi$ 走一步」翻译成「沿曲面（$SO(3)$）测地线绕轴 $\mathbf{u}$ 转 $\theta$ 角」。它保证结果一定是合法旋转矩阵。

### 4.4 对数映射：李群 → 李代数

反方向用 $\log$，从旋转矩阵恢复轴角:

$$\theta = \arccos\!\left(\frac{\operatorname{tr}(R)-1}{2}\right),\qquad \boldsymbol\phi^{\wedge} = \log(R) = \frac{\theta}{2\sin\theta}\,(R - R^{\top})$$

（$\theta\to0$ 时用泰勒展开避免除零；$\theta\to\pi$ 是奇异点，要特殊处理。）

### 4.5 李括号：李代数的「乘法」

李代数除了向量加法，还有一个内禀运算 **李括号（Lie bracket）**，对矩阵就是对易子:

$$[A, B] = AB - BA$$

对 $\mathfrak{so}(3)$，它恰好对应叉乘：$[\boldsymbol\phi_1^\wedge, \boldsymbol\phi_2^\wedge] = (\boldsymbol\phi_1\times\boldsymbol\phi_2)^\wedge$。李括号度量了「两个旋转不可交换的程度」——先转 A 再转 B，和先转 B 再转 A，差别由李括号刻画（这也是 BCH 公式的一阶项，见 §六）。

### 4.6 $\mathfrak{se}(3)$：6 维 twist

$SE(3)$ 的李代数是 6 维向量 $\boldsymbol\xi=(\boldsymbol\rho,\ \boldsymbol\phi)\in\mathbb{R}^6$，称为 **twist**（运动旋量），$\boldsymbol\rho$ 关乎平移、$\boldsymbol\phi$ 关乎旋转。其 hat 形式是 $4\times4$ 矩阵:

$$\boldsymbol\xi^{\wedge} = \begin{pmatrix} \boldsymbol\phi^{\wedge} & \boldsymbol\rho \\ \mathbf{0}^\top & 0 \end{pmatrix}\in\mathbb{R}^{4\times4}$$

指数映射 $\exp(\boldsymbol\xi^\wedge)=\begin{pmatrix} R & J\boldsymbol\rho\\ \mathbf{0}^\top & 1\end{pmatrix}$，其中 $R=\exp(\boldsymbol\phi^\wedge)$，而 $J$ 是**左雅可比**:

$$J = I + \frac{1-\cos\theta}{\theta^2}\boldsymbol\phi^{\wedge} + \frac{\theta-\sin\theta}{\theta^3}(\boldsymbol\phi^{\wedge})^2$$

> **注意一个常见误区**：$\boldsymbol\xi$ 里的平移部分 $\boldsymbol\rho$ **不等于** 最终位姿的平移 $\mathbf{t}=J\boldsymbol\rho$。因为旋转和平移在 $SE(3)$ 上是耦合的（边转边走，轨迹是螺旋线）。只有纯平移（$\boldsymbol\phi=0$，此时 $J=I$）时两者才相等。

### 4.7 小结：两座桥

```
        李代数 (so(3)/se(3))                李群 (SO(3)/SE(3))
        平直向量空间 ℝ³ / ℝ⁶               弯曲流形
        ─────────────────                 ─────────────────
        可加减、可求导、可优化      exp →    合法旋转/位姿、可作用于点
        (角速度/旋转向量/twist)     ← log    (朝向/位姿)

        优化范式：在李代数里算增量 δ  ──exp──▶  更新李群上的状态
```

## 五、核心应用：为什么优化要在流形上做

这是整套理论在具身智能里最重要的「为什么」。

### 5.1 问题：不能对旋转直接做梯度下降

SLAM、Bundle Adjustment、6DoF 位姿估计、IK 的本质都是**优化一个位姿**使某个误差最小，比如重投影误差 $\min_R \sum_i \| \pi(R\mathbf{p}_i) - \mathbf{u}_i\|^2$。梯度下降需要「当前估计 + 小步增量」，但:

- 直接更新 $R \leftarrow R + \Delta$ → 结果不再是旋转矩阵（破坏正交性）。
- 用欧拉角参数化 → 遇到万向锁，雅可比奇异，优化卡死。

### 5.2 解法：⊞ / ⊟（boxplus / boxminus）

标准做法是把增量放在**李代数**（平直空间）里，用 $\exp$ 映射回流形再左乘/右乘:

$$\boxed{\,R \boxplus \boldsymbol\delta \;:=\; R\cdot\exp(\boldsymbol\delta^{\wedge}),\qquad \boldsymbol\delta\in\mathbb{R}^3\,}$$

$$R_1 \boxminus R_2 \;:=\; \log(R_2^{-1}R_1)^{\vee}\in\mathbb{R}^3$$

- $\boldsymbol\delta$ 活在平直的 $\mathbb{R}^3$ 里，可以自由求雅可比、做高斯牛顿 / LM 迭代。
- $\exp$ 保证更新后 $R$ 仍严格是合法旋转，**永不漂出流形**。
- $\boxminus$ 给出两个旋转之间真正的「流形上差异」（可当作误差/距离）。

迭代一步长这样:

$$R^{(k+1)} = R^{(k)} \boxplus \boldsymbol\delta^{*},\qquad \boldsymbol\delta^{*} = -(J^\top J)^{-1}J^\top \mathbf{r}$$

其中 $\mathbf{r}$ 是残差，$J=\partial \mathbf{r}/\partial\boldsymbol\delta$ 是残差对**李代数增量**的雅可比。

> **这就是现代后端优化的统一范式。** Ceres、g2o、GTSAM、Sophus、`manif`、`liegroups` 这些库，本质都在帮你做「在 $\mathfrak{so}(3)/\mathfrak{se}(3)$ 里求增量 → `exp` 回 $SO(3)/SE(3)$」。理解了 ⊞/⊟，就理解了它们的内核。

### 5.3 运动学：把角速度积分成朝向

这是 IMU / 状态估计的命脉。连续时间下，朝向的导数与角速度的关系（**泊松方程**）:

$$\dot R = R\,\boldsymbol\omega_b^{\wedge}\quad(\text{机体系角速度})\qquad\text{或}\qquad \dot R = \boldsymbol\omega_w^{\wedge}R\quad(\text{世界系角速度})$$

离散化后，每个时间步用 $\exp$ 更新（陀螺仪给出 $\boldsymbol\omega$）:

$$R_{k+1} = R_k\,\exp\!\big((\boldsymbol\omega_k\,\Delta t)^{\wedge}\big)$$

这正是 VIO / IMU 预积分（Forster et al. 2017）的核心步骤：把一段时间的陀螺仪读数在 $\mathfrak{so}(3)$ 里累加、`exp` 回 $SO(3)$，得到相对旋转约束。

## 六、进阶：伴随、雅可比、BCH（按需了解）

这一节是「知道有这回事」即可，初读可跳过，做 SLAM/标定推导时再回来。

- **伴随表示 $\mathrm{Ad}$**：在不同坐标系之间搬运 twist。$R\,\exp(\boldsymbol\phi^\wedge) = \exp((R\boldsymbol\phi)^\wedge)\,R$，即 $\mathrm{Ad}_R=R$。换坐标系时左扰动 ↔ 右扰动的转换全靠它。
- **左/右雅可比 $J_l, J_r$**：刻画 $\exp$ 的非线性，使「李代数的微小变化」与「李群上的微小变化」精确对应，是误差状态卡尔曼滤波（ESKF）和因子图求导的必备。
- **BCH 公式**：$\exp(A)\exp(B)=\exp\big(A+B+\tfrac12[A,B]+\cdots\big)$。它解释了**为什么旋转不可交换**——多出来的 $\tfrac12[A,B]$（李括号）就是不可交换的代价；当旋转很小（一阶近似）时可忽略，于是「小旋转近似可加」。

## 七、四元数：表示 $SO(3)$ 最实用的坐标

旋转矩阵冗余（9 个数）、欧拉角有万向锁、轴角有 $2\pi$ 奇异。工程上存储和传播旋转，**单位四元数**是综合最优的选择。

### 7.1 定义与乘法

四元数是复数的推广，有 1 个实部 + 3 个虚部:

$$q = w + x\,\mathbf{i} + y\,\mathbf{j} + z\,\mathbf{k},\qquad \mathbf{i}^2=\mathbf{j}^2=\mathbf{k}^2=\mathbf{ijk}=-1$$

由此推出 $\mathbf{ij}=\mathbf{k},\ \mathbf{jk}=\mathbf{i},\ \mathbf{ki}=\mathbf{j}$（循环，且**不可交换**：$\mathbf{ji}=-\mathbf{k}$）。常把它写成「标量 + 向量」$q=(w,\ \mathbf{v})$，$\mathbf{v}=(x,y,z)$。

**Hamilton 乘积**（两个旋转的复合）:

$$q_1 q_2 = \big(\,w_1w_2 - \mathbf{v}_1\cdot\mathbf{v}_2,\ \ w_1\mathbf{v}_2 + w_2\mathbf{v}_1 + \mathbf{v}_1\times\mathbf{v}_2\,\big)$$

那个 $\mathbf{v}_1\times\mathbf{v}_2$ 叉乘项，正是四元数乘法不可交换的来源——和旋转不可交换是同一件事。

> ⚠️ **工程大坑**：四元数有两种约定。**Hamilton**（$\mathbf{ij}=\mathbf{k}$，ROS、Eigen、Ceres、g2o）vs **JPL**（$\mathbf{ij}=-\mathbf{k}$，部分航天/滤波文献）。两者乘法顺序和符号不同，混用会让旋转方向反掉。**始终确认你的库用哪种**（绝大多数机器人栈是 Hamilton）。另一个坑是存储顺序：Eigen 内部存 $(x,y,z,w)$ 但构造函数是 `Quaterniond(w,x,y,z)`。

### 7.2 用四元数表示旋转

绕**单位轴** $\mathbf{u}$ 转 $\theta$ 角的旋转，对应单位四元数:

$$\boxed{\,q = \Big(\cos\tfrac{\theta}{2},\ \ \mathbf{u}\,\sin\tfrac{\theta}{2}\Big),\qquad \|q\|=1\,}$$

注意那个**半角 $\theta/2$**——它是「双重覆盖」的根源（见 7.4）。

**旋转一个向量** $\mathbf{p}$：把 $\mathbf{p}$ 看成纯四元数 $(0,\mathbf{p})$，则

$$\mathbf{p}' = q\,(0,\mathbf{p})\,q^{-1},\qquad \text{单位四元数的逆 } q^{-1}=\bar q=(w,-\mathbf{v})\ (\text{共轭})$$

**复合**：先 $q_2$ 后 $q_1$ = $q_1q_2$，和旋转矩阵 $R_1R_2$ 完全对应。

### 7.3 为什么工程偏爱四元数

| 优点 | 说明 |
|------|------|
| 紧凑 | 4 个数 vs 矩阵 9 个数 |
| 无万向锁 | 不像欧拉角会丢自由度 |
| 数值稳定 | 长期复合后只需重新归一化 $q/\|q\|$，矩阵则要做代价更高的正交化（SVD/Gram-Schmidt） |
| 复合快 | Hamilton 乘积比 $3\times3$ 矩阵乘法运算量小 |
| 插值好 | SLERP 给出匀速最短路径插值（见 7.5） |

### 7.4 双重覆盖：$q$ 与 $-q$ 是同一个旋转

因为是半角，$q=(\cos\frac\theta2, \mathbf{u}\sin\frac\theta2)$ 和 $-q=(\cos\frac{\theta+2\pi}{2}, \mathbf{u}\sin\frac{\theta+2\pi}{2})$ 表示**同一个旋转**。即单位四元数集合 $S^3$（四维单位球面）以 **2:1** 的方式覆盖 $SO(3)$。

- 拓扑上：单位四元数群 $=SU(2)$，它是 $SO(3)$ 的**双重覆盖**，$S^3\to SO(3)$ 是 2:1 同态。
- **实践影响**：(1) 比较/求平均/插值两个四元数前，先统一符号（取点积为正的一支，否则会走 $360°$ 的远路）；(2) 神经网络直接回归四元数会因 $\pm q$ 的二义性产生**不连续**（见 §八）。

### 7.5 SLERP：四元数球面插值

给定起止朝向 $q_0,q_1$，要在它们之间**匀速、走最短弧**地插值（做轨迹平滑、动画、姿态规划）:

$$\mathrm{slerp}(q_0,q_1;t) = \frac{\sin\big((1-t)\Omega\big)}{\sin\Omega}\,q_0 + \frac{\sin(t\Omega)}{\sin\Omega}\,q_1,\qquad \cos\Omega = q_0\cdot q_1,\ t\in[0,1]$$

它沿 $S^3$ 上的大圆（测地线）等速移动，等价于在 $\mathfrak{so}(3)$ 里直线插值再 `exp` 回去。$\Omega$ 很小时退化为普通线性插值（Nlerp）。

### 7.6 四元数与李代数的关系

四元数也有自己的 `exp`/`log`。对小旋转 $\boldsymbol\phi=\theta\mathbf{u}\in\mathfrak{so}(3)$:

$$\exp_{\mathrm{quat}}(\boldsymbol\phi) = \Big(\cos\tfrac\theta2,\ \mathbf{u}\sin\tfrac\theta2\Big)$$

其李代数仍是 $\mathfrak{so}(3)\cong\mathbb{R}^3$（纯虚四元数）。最有用的是**四元数运动学方程**（IMU 积分的四元数版，比矩阵版省算）:

$$\dot q = \tfrac12\, q\otimes(0,\ \boldsymbol\omega_b)\qquad\Longrightarrow\qquad q_{k+1} = q_k\otimes\exp_{\mathrm{quat}}\!\big(\tfrac12\boldsymbol\omega_k\Delta t\big)$$

## 八、深度学习里的旋转表示：连续性与 6D 表示

具身智能的策略网络（VLA、Diffusion Policy、姿态回归）常要**让网络输出一个旋转**。这里有个不直观但关键的结论。

### 8.1 为什么不能直接回归四元数 / 欧拉角

Zhou et al.（CVPR 2019, *On the Continuity of Rotation Representations in Neural Networks*）指出:

> **$SO(3)$ 在任何 $\le 4$ 维的实欧氏空间里的表示都是不连续的。**

直觉：$SO(3)$ 的拓扑（它是 $\mathbb{RP}^3$）无法被一张 $\le4$ 维的连续无缝「地图」覆盖。具体表现:

- **欧拉角**：在 $\pm\pi$ 处跳变，且万向锁。
- **四元数**：$q$ 和 $-q$ 同义，网络在边界附近被迫从 $q$ 跳到 $-q$，输出曲面被撕开一道缝；回归目标不连续 → 网络在这些区域误差大、难收敛。
- **轴角**：在 $\theta=\pi$ 附近不连续。

不连续意味着「输入平滑变化时，理想输出要突变」，神经网络（连续函数逼近器）天生学不好这种突变。

### 8.2 6D 表示（连续，现已主流）

解法是用**冗余但连续**的表示。最常用的是 **6D 表示**：网络输出 6 个数 = 两个 3 维向量 $\mathbf{a}_1,\mathbf{a}_2$，再用 Gram-Schmidt 正交化恢复旋转矩阵:

$$\mathbf{b}_1=\frac{\mathbf{a}_1}{\|\mathbf{a}_1\|},\quad \mathbf{b}_2=\frac{\mathbf{a}_2-(\mathbf{b}_1\!\cdot\!\mathbf{a}_2)\mathbf{b}_1}{\|\cdots\|},\quad \mathbf{b}_3=\mathbf{b}_1\times\mathbf{b}_2,\quad R=[\,\mathbf{b}_1\ \mathbf{b}_2\ \mathbf{b}_3\,]$$

这个映射处处连续，网络好学。Diffusion Policy、π₀ 等动作头的旋转分量、各类 6DoF 姿态回归现在大量用 6D（也有用 9D / SVD 正交化的变体）。

> **一句话经验**：**存储/传播用四元数，网络输出用 6D，优化增量用 so(3)。** 各取所长。

## 九、表示之间的转换（速查）

```
                    ┌─────────────┐
            log     │  so(3) = ℝ³ │   exp(Rodrigues)
        ┌──────────▶│  旋转向量φ    │◀──────────┐
        │           └─────────────┘            │
        │             θ/2, u │  半角            │
        │                    ▼                  │
  ┌───────────┐   q↔R   ┌──────────┐    Gram-Schmidt   ┌────────┐
  │ 旋转矩阵 R │◀───────▶│ 四元数 q  │       (取两列) ◀──│ 6D 表示 │
  └───────────┘         └──────────┘                   └────────┘
        │   ▲
        │   │ 欧拉角(有万向锁，少用于计算)
        ▼   │
   ┌─────────────┐
   │ roll-pitch-yaw│
   └─────────────┘
```

**四元数 → 旋转矩阵**（$q=(w,x,y,z)$，已归一化）:

$$R = \begin{pmatrix} 1-2(y^2+z^2) & 2(xy-wz) & 2(xz+wy)\\ 2(xy+wz) & 1-2(x^2+z^2) & 2(yz-wx)\\ 2(xz-wy) & 2(yz+wx) & 1-2(x^2+y^2)\end{pmatrix}$$

工程上不用手推，几行代码搞定:

```python
import numpy as np
from scipy.spatial.transform import Rotation as Rt

# 轴角 → 旋转矩阵 / 四元数（scipy 用 [x,y,z,w] 顺序存四元数）
rotvec = np.array([0, 0, np.pi/2])          # 绕 z 轴转 90°
R  = Rt.from_rotvec(rotvec).as_matrix()      # exp:  so(3) → SO(3)
q  = Rt.from_rotvec(rotvec).as_quat()        # → 四元数
phi = Rt.from_matrix(R).as_rotvec()          # log:  SO(3) → so(3)

# 复合：先 R2 后 R1
R12 = Rt.from_matrix(R1.as_matrix() @ R2.as_matrix())

# SLERP 插值（轨迹平滑常用）
from scipy.spatial.transform import Slerp
slerp = Slerp([0, 1], Rt.from_quat([q0, q1]))
q_mid = slerp(0.5).as_quat()
```

> PyTorch 端做可微旋转用 `pytorch3d.transforms`（`matrix_to_rotation_6d`、`rotation_6d_to_matrix`、`so3_exp_map`、`quaternion_multiply` 等），训练 VLA / 姿态网络时直接用。

## 十、回到具身智能：六个落地场景

| 场景 | 用到的数学 | 关键点 |
|------|-----------|--------|
| **IMU / VIO 状态估计** | $\mathfrak{so}(3)$ 积分 + `exp` | 陀螺仪角速度在李代数累加，`exp` 回 $SO(3)$；IMU 预积分给出相对位姿因子 |
| **视觉 SLAM / BA** | $SE(3)$ 上的 ⊞/⊟ 优化 | 相机位姿在流形上做高斯牛顿，雅可比对 $\mathfrak{se}(3)$ 增量求；g2o/GTSAM/Ceres |
| **6DoF 抓取 / 位姿估计** | $SE(3)$ 表示 + 损失 | 预测夹爪位姿；旋转损失用测地线距离 $\|\log(R_{gt}^\top R_{pred})\|$ 而非欧氏 |
| **Diffusion Policy / VLA 动作** | 6D 旋转表示 | 末端位姿增量 = 平移 $\mathbb{R}^3$ + 旋转 6D；连续表示让动作头好训 |
| **人体动作 / SMPL** | SO(3) 轨迹 | 每个关节是 $SO(3)$ 旋转（轴角参数化），一段动作 = $SO(3)^J$ 上的时间序列；motion 生成/重定向/IK 全在这套语言里 |
| **逆运动学 / 运动规划** | $SE(3)$ 配置空间 | 末端目标在 $SE(3)$，雅可比把关节速度映到 twist；流形上插值得到平滑轨迹 |

> **与人体中心方向的连接**：human-centric embodied AI / motion 这条线尤其离不开 $SO(3)$——SMPL/SMPL-X 的 body pose 就是每个关节相对父关节的旋转，动作生成、retargeting、物理仿真里的损失函数、插值、积分，本质都在 $SO(3)$ 上做。把这套数学吃透，是做 motion / 人体姿态方向的基本功。

## 十一、常见坑与面试问答

**Q1. 为什么不能直接对旋转矩阵做梯度下降 / 求平均？**
因为 $SO(3)$ 不是向量空间，逐元素加法/平均会破坏正交约束，结果不再是旋转。要在 $\mathfrak{so}(3)$ 里做增量再 `exp` 回去（⊞），求平均用 chordal/geodesic mean。

**Q2. 万向锁（gimbal lock）到底是什么？**
欧拉角连续旋转时，某个中间角到 $\pm90°$ 会让两个旋转轴重合，丢失一个自由度，雅可比奇异。四元数/旋转矩阵/轴角都没有这个问题——这是用 3 个角度参数化的固有缺陷。

**Q3. 四元数为什么是「半角」？$q$ 和 $-q$ 什么关系？**
半角来自 $S^3$ 对 $SO(3)$ 的 2:1 双重覆盖；$q$ 与 $-q$ 是同一个旋转。比较/插值/求平均前要先统一符号（令点积为正）。

**Q4. 深度网络输出旋转，用四元数还是 6D？**
优先 6D（或 9D+SVD）。四元数/欧拉角在 $SO(3)$ 上不连续（Zhou 2019），网络难学边界区域。存储可以转回四元数。

**Q5. 旋转误差/损失该怎么定义？**
用测地线距离 $d(R_1,R_2)=\|\log(R_1^\top R_2)^\vee\| = \theta$（相对旋转角），而不是矩阵的 Frobenius 距离或四元数的欧氏差——后者不是流形上的真实距离。

**Q6. so(3) 的 twist 平移分量 $\boldsymbol\rho$ 等于位姿平移 $\mathbf{t}$ 吗？**
不等于，$\mathbf{t}=J\boldsymbol\rho$（$J$ 是左雅可比）。只有纯平移时相等。这是 $\mathfrak{se}(3)$ 的常见误解。

**Q7. Hamilton 和 JPL 四元数的区别？**
虚数单位乘法约定不同（$\mathbf{ij}=\mathbf{k}$ vs $\mathbf{ij}=-\mathbf{k}$），导致乘法顺序、旋转方向不同。机器人栈（ROS/Eigen/Ceres）几乎都是 Hamilton；用前务必确认，否则旋转会反。

## 十二、一页速查总结

| 概念 | 是什么 | 在具身智能里的角色 |
|------|--------|---------------------|
| **$SO(3)$** | 3D 旋转群（正交、$\det{=}1$ 矩阵），3 维弯曲流形 | 朝向住的空间 |
| **$SE(3)$** | 刚体变换群（$4\times4$ 齐次矩阵），6 维 | 完整位姿住的空间 |
| **$\mathfrak{so}(3)$** | 反对称矩阵 ≅ $\mathbb{R}^3$，= 旋转向量/角速度 | 旋转增量、角速度积分 |
| **$\mathfrak{se}(3)$** | 6 维 twist $(\boldsymbol\rho,\boldsymbol\phi)$ | 位姿增量、运动旋量 |
| **exp / log** | 李代数 ↔ 李群的桥（Rodrigues 公式） | 增量映射回流形 / 从位姿取增量 |
| **⊞ / ⊟** | 流形上的「加 / 减」 | SLAM、BA、滤波的优化原语 |
| **四元数** | $S^3$ 上单位向量，2:1 覆盖 $SO(3)$ | 紧凑存储、复合、SLERP 插值、IMU 姿态 |
| **SLERP** | 球面匀速最短弧插值 | 轨迹/动画/姿态平滑 |
| **6D 表示** | 两列向量 + Gram-Schmidt | 神经网络输出旋转（连续、好训） |

**一条主线收尾**：

> $SO(3)/SE(3)$ 是姿态真正住的**弯曲流形**（李群）；$\mathfrak{so}(3)/\mathfrak{se}(3)$ 是它在单位元处的**平直切空间**，让你能加减、求导、优化（李代数）；二者通过 **exp / log** 互换。**四元数**是 $SO(3)$ 一套紧凑、无万向锁、易插值的实用坐标；**6D 表示**是为神经网络量身定做的连续坐标。它们共同支撑具身智能里一切关于「转动与位姿」的表示、积分、插值与优化。
