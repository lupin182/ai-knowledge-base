# 战略主线与技术拐点（2026）

> 基于 2026 Q2 行业状态。本文把 2025-2026 的具身智能拆成 **三条技术主线 + 两条产业主线**，并梳理 2026 年四大技术拐点与关键时间表。视角兼顾技术研究者与产业观察者。

---

```
  ─────────────────────────────────────

  灵巧手: PaXini DexH13, Inspire RH56DFX, Figure Hand v2
  触觉: GelSight, Digit 360, BioTac, PaXini 触觉阵列
```

### 1.2 两条产业主线


## 二、2026 四大技术拐点

### 拐点 1：VLA 从"像 LLM"变成"像机器人"

2023-2024 年的 VLA（RT-2、OpenVLA）把动作当 token 直接吐，本质上是"LLM + 动作词表"。2025-2026 年彻底转向 **连续动作头 + 异步双频**，VLM 和 Action Expert 解耦。

| 阶段 | 动作表示 | 代表 | 控制频率 |
|------|---------|------|---------|
| 2023 | 离散 token（256 桶） | RT-2, OpenVLA | 1-5 Hz |
| 2024 | 扩散头 / Flow Matching | π₀, Octo, RDT-1B | 10-30 Hz |
| 2025 | 动作专家网络 + VLM 分离 | GR00T N1, Helix, π₀.5 | 30-200 Hz |
| 2026 | 异步双频 + 长 horizon chunking | π₀.5+, GR00T N1.5, Gemini-ER | 双频 30-200 Hz |

**核心特征**：

- **VLM 慢**（7-9 Hz）：输出高层 latent / language plan，保留常识与推理
- **Action Expert 快**（100-200 Hz）：Flow Matching / Diffusion / Transformer，生成连续动作 chunk
- **Chunking**：一次预测未来 1-2 秒 50 步动作，降低推理压力
- **VLM 半冻结**：保留预训练能力，微调动作头为主

Figure Helix 的 S2 (7B VLM @ 9Hz) + S1 (80M Transformer @ 200Hz) 是该范式的代表。详见 [../01-VLA模型/](../01-VLA模型/)。

### 拐点 2：世界模型成为训练的"第三条腿"

2025 年 NVIDIA Cosmos、Google Genie 2/3、Meta V-JEPA 2、1X World Model 集体发布。**世界模型从"视频生成玩具"变成"机器人数据/仿真基础设施"**。

```
三源训练数据 (2026 头部公司标配):

    真实遥操作轨迹 ─┐
                    │
    物理仿真数据   ─┼──→ VLA / Policy 训练
                    │
    世界模型合成   ─┘
    （Cosmos / Genie / 1X WM / V-JEPA 2）
```

**世界模型的三个具身用途**：

1. **数据增广**：一条真实轨迹 → N 条新视角 / 新光照 / 新物体的合成轨迹（Cosmos、GR00T Dreams）
2. **Policy rollout 评估**：在 WM 中快速筛选策略版本，取代部分真机测试
3. **隐式规划 / 状态估计**：V-JEPA 2 在隐空间做 zero-shot planning

详见 [../02-世界模型/](../02-世界模型/)。

### 拐点 3：灵巧手从"研究玩具"变为"量产部件"

2024 年灵巧手以 Allegro / LEAP / Shadow 为代表，单价 $30-100K 的研究硬件。2025-2026 出现第一波 **规模量产的商用灵巧手**：

| 厂商 | 型号 | 主动自由度 | 单价（据披露） | 特点 |
|------|------|-----------|--------------|------|
| Tesla | Optimus Hand Gen 2 | 22 | — | 腱绳 + 本体感知 |
| Figure | Figure Hand v2 | 16 | — | 与 Helix 配套 |
| 因时机器人 | Inspire RH56DFX | 12 | ~¥3 万 | 国产量产规模最大 |
| PaXini | DexH13 | 13 | — | 触觉阵列集成 |
| Sanctuary | Phoenix Hand | 21 | — | 液压驱动 |

**工程难点**：非降维全驱动、电机/腱绳的背部集成、触觉融合。2026 年 **触觉传感**（GelSight、Digit 360、BioTac、PaXini）开始大规模整合进量产手，视觉+触觉 VLA 正在成形。详见 [../08-硬件与本体/](../08-硬件与本体/)。

### 拐点 4：跨本体泛化（Cross-Embodiment）成立

过去"每换一个机器人就要重训策略"的桎梏在 2025-2026 被实质性打破：

- **Open X-Embodiment**（2023，22 种机器人, 160K+ episodes）奠定跨本体数据基础
- **AgiBot World**（2024，智元 100 万+ 条真机轨迹）成为国内最大跨本体集
- **DROID**（76K 多机器人操控）
- **π₀** 在 8+ 种本体（Franka、UR5、ARX、ALOHA、YAM…）上验证跨本体训练
- **Gemini Robotics-ER** 号称 zero-shot 迁移到新本体

**2026 新趋势**：**本体无关的潜在动作空间**（Latent Action Space）——模型输出抽象动作表示，由本体适配层映射到具体关节。这条路径有望把"单本体数据"变成"全行业可复用资产"。

## 三、大脑-小脑界面的重新划分

### 3.1 三代架构演进

```
第一代 (2022-2023):  VLM 规划 + 技能库执行
  ┌──────────┐    subgoal    ┌──────────────┐
  │   VLM    │ ───────────→ │  技能库 (Pre- │
  │ (SayCan) │               │  trained Skills)│
  └──────────┘               └──────────────┘
  问题: 技能库有限、不易扩展、能力上限受限

第二代 (2024):  纯端到端 VLA
  ┌──────────────────────────────────┐
  │       VLA (RT-2 / OpenVLA)        │
  │  视觉+语言 → 动作 token (1-5Hz)    │
  └──────────────────────────────────┘
  问题: 频率过低、无多模态建模、长 horizon 差

第三代 (2025-2026 共识):  双系统 VLA + 可插拔慢思考
  ┌─────────────────────────┐
  │  慢思考 (可选)            │  ← LLM 规划、语言子目标、Code-as-Policies
  │  (VLM 推理 / Gemini-ER)   │
  └───────────┬─────────────┘
              │ language / subgoal
              ▼
  ┌─────────────────────────┐
  │  VLM Backbone  (7-9 Hz) │  ← System 2, "做什么"
  └───────────┬─────────────┘
              │ latent / token
              ▼
  ┌─────────────────────────┐
  │  Action Expert (100-200Hz)│  ← System 1, "怎么做"
  └───────────┬─────────────┘
              │ 连续动作 chunk
              ▼
  ┌─────────────────────────┐
  │  本体控制（Adapter / WBC） │
  └─────────────────────────┘
```

### 3.2 核心共识：端到端 VLA 主干 + 可插拔慢思考

2026 行业共识 = **端到端双频 VLA 作为主干，慢思考（长 horizon 规划 / 多步推理 / 代码生成）作为可插拔上层**。

- 主干代表：Figure Helix、π₀.5、GR00T N1.5
- 慢思考代表：Gemini Robotics-ER（推理版，不输出动作）、SayCan 类、Code as Policies
- 工业部署时慢思考常常被替换为 **任务脚本 + 参数化调用**，C 端任务才真正需要完整慢思考

## 五、小结：2026 的结构性判断

1. **大脑架构收敛**：异步双频 VLA 是 2026 的主流语言，争论点已从"是否要双系统"变为"VLM 多大、Action Head 怎么做、chunk 多长"。
2. **世界模型成为基础设施**：从视频生成工具变为机器人训练/评估 pipeline 的必备组件。
3. **灵巧手、触觉、端侧芯片** 是 2026-2027 的硬件三支柱，决定 C 端时间表。
4. **跨本体数据** 是 2026 最深的护城河——谁汇聚数据，谁掌握长期复利。
5. **B 端 2025-2027 验证 ROI，C 端 2027+ 试水**，时间表比乐观派至少慢 1-2 年。
