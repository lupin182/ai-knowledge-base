# 主要玩家与战略定位（2026）

> 本文梳理 2026 年具身智能的主要玩家：**美国头部 + 中国头部 + 科技巨头战略布局**，并给出"垂直整合 vs 水平分层"的四象限战略地图。融资/估值数据标注"据披露"，时间以 2026 Q2 为基准。

---

## 一、美国头部公司

### 1.1 Physical Intelligence（π₀ / π₀.5 / π₀.7）

```
路线:     本体无关的基础模型，B2B 技术赋能
团队:     Sergey Levine (Berkeley)、Chelsea Finn (Stanford)、Karol Hausman (前 Google)
融资:     据披露 2024 B 轮 $400M（红杉、OpenAI 领投），估值 $2-3B
里程碑:   π₀ (2024.10) → π₀.5 (2025.04, 家庭泛化) → π-fast / π₀.7 (2025H2-2026)
2026 重点: 
  · π₀.5 家庭长 horizon 泛化
  · π-fast 实时化部署（30Hz+）
  · 多本体伙伴网络持续扩张（8+ → 15+）
核心壁垒: 跨本体数据 + 模型人才密度
```

### 1.2 Figure AI

```
路线:     垂直整合（硬件 + 模型），B 端商业落地优先
团队:     Brett Adcock（连续创业者）
融资:     据披露 2024 B 轮 $675M（微软、OpenAI、NVIDIA 等），估值 $2.6B
          2025-2026 进一步融资进行中（据披露估值目标 $39.5B+）
里程碑:   Figure 01 → 02 (2024 @ BMW) → Helix VLA (2025.02) → Figure 03 (2025.10)
2026 重点:
  · 千台级产能爬坡
  · 家庭场景 Beta（Figure 03）
  · Helix v2 长 horizon 能力
核心壁垒: 硬件-软件协同 + BMW 商业数据闭环
```

### 1.3 1X Technologies

```
路线:     家用陪伴机器人 Neo，世界模型驱动
特色:     半人高 + 软包覆（安全性优先），订阅制
融资:     据披露 OpenAI Startup Fund 投资，多轮融资
里程碑:   EVE → Neo → Neo Gamma (2025)
商业模式: Neo Home 订阅，据披露约 $20K / 年
2026 重点: Neo 家庭 Beta 测试规模化
核心壁垒: 家用场景数据 + 1X World Model
```

### 1.4 Tesla Optimus

```
路线:     垂直整合 + 超大规模自产，对标 Model 3 降本
迭代:     Gen 1 (2022) → Gen 2 (2024.10) → Gen 3 (预计 2026)
产能目标: 据披露 2026 内部 5000+ 台，2027 对外销售
核心优势:
  · 供应链（电池、电机、制造）复用
  · FSD 的数据与算力架构复用
  · Dojo / HW 系列自研芯片
风险: 马斯克承诺兑现节奏、Optimus 与 FSD 资源争夺
```

### 1.5 Agility Robotics（Digit）

```
路线:     物流专用双足，已商业化
产品:     Digit v4 → v5
部署:     GXO Logistics（首个商业客户）、Amazon 仓库
融资:     据披露 2024 $150M C 轮
2026 重点: Digit v5，从"搬箱"扩展到"多品类"
核心壁垒: 最早在仓储场景实际出货的人形双足
```

### 1.6 Apptronik（Apollo）

```
背景:     NASA Valkyrie 团队商业化（德州大学奥斯汀分校）
合作:     Mercedes-Benz（工业试点）、Google DeepMind（模型）
融资:     据披露 2024 $350M A 轮（B Capital、Capital Factory 领投）
2026 重点: 与 DeepMind Gemini Robotics 共研，Apollo 工业部署
核心壁垒: NASA 级工程背景 + DeepMind 独家合作
```

### 1.7 Boston Dynamics

```
路线:     电动新 Atlas（2024.4 发布），全电驱、无液压
所有者:   现代汽车集团
产业:     现代工厂试点
战略:     与 RAI Institute（原 BD AI Institute，Marc Raibert 牵头）分工
          · BD: 本体 + 商业化
          · RAI: 前沿研究
核心壁垒: 二十年 locomotion / WBC 积累
```

### 1.8 Sanctuary AI（Phoenix）

```
特色:     液压驱动 + 高精度灵巧手（21-DOF）
定位:     偏研究，少量零售/物流试点
2026 重点: 通用 AI 优先于量产，节奏较慢
```

### 1.9 Skild AI / Covariant / Dexterity 等"软件层"

| 公司 | 路线 | 关键事件 |
|------|------|---------|
| Skild AI | 通用机器人基础模型 | 据披露 2024.7 $4.5B 估值，软银 / 亚马逊 / NVIDIA 投资 |
| Covariant | 工业操控基础模型 | 2024 年被亚马逊"acquihire"，核心团队加入 Amazon |
| Dexterity | 物流操作 AI | 工业客户 + 多轮融资 |
| Generalist AI（Pieter Abbeel） | 通用仿真 + 本体 | 早期，对标 Cosmos / UniSim |

## 四、垂直整合 vs 水平分层：四象限战略地图

### 4.1 战略路线对比表

| 路线 | 代表 | 核心资产 | 优势 | 劣势 |
|------|------|---------|-----|-----|
| **垂直整合** | Figure、Tesla、1X、Optimus、智元 | 硬件 + 模型 + 数据 | 协同优化，数据闭环强 | 重资产、资本密集 |
| **水平分层：模型层** | Physical Intelligence、Skild、Covariant、星动、千寻 | 跨本体基础模型 | 轻资产、覆盖多本体 | 依赖硬件伙伴，数据采集被动 |
| **水平分层：平台层** | NVIDIA（Isaac/Cosmos/GR00T）、华为（昇腾/盘古）、Google（Gemini） | 工具链 + 生态 + 芯片 | 工具链 + 标准 + 渠道 | 需要生态繁荣才变现 |
| **水平分层：本体层** | 宇树、傅利叶、EngineAI、Agility、Apptronik | 硬件规模化制造 | 本体规模化、成本下行 | 大脑依赖第三方，议价低 |

### 4.3 2026 的共识与争议

**共识**：
- **短期（2025-2027）**：垂直整合头部领跑 B 端商业化（Figure、Tesla、智元、优必选）
- **长期（2028+）**：水平模型层可能胜出，类比 Android/iOS 与自营手机的关系——但是否真会出现，仍有争议

**争议**：
- 模型层是否能在"缺少本体 + 数据自主权"的情况下维持领先？
- 平台方（NVIDIA）是否会向上延伸做模型 / 本体（参考 Apple Silicon 路径）？
- 本体层能否向上整合做模型（宇树正在尝试）？

## 五、端到端 vs 分层：另一个维度

| 路线 | 代表 | 能力上限 | 数据需求 | 可解释性 |
|------|------|---------|---------|---------|
| **纯端到端 VLA** | π₀、Helix | 理论上限高 | 极大 | 低 |
| **分层（VLM + 技能库）** | SayCan 系、早期 GR00T、工业部署 | 受技能库限制 | 较小 | 高 |
| **混合双系统（2026 主流）** | Helix、π₀.5、GR00T N1.5、Gemini + Apollo | 兼顾 | 中 | 中 |

**2026 实际落地模式**：B 端工业部署往往使用"双系统 VLA + 任务参数化脚本"混合，C 端家庭场景才真正使用完整端到端 + 慢思考。

---

## 六、小结：2026 产业格局的四个观察

1. **Figure、Tesla、1X 三家**代表美国垂直整合头部，资金/数据/品牌优势最大
2. **Physical Intelligence + NVIDIA + DeepMind** 组成模型/平台铁三角，生态影响力最大
3. **智元 / 宇树 / 星动 / 优必选** 是中国头部代表，数据工厂 + 供应链成本是最强护城河
4. **OpenAI / Microsoft / Apple / Amazon** 从投资方/云方/渠道方参与，未来 1-2 年可能有重磅动作（OpenAI 自研人形）
