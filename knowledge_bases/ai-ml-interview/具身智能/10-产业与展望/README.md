## 一、2026 产业发展现状

### 1.1 从"实验室"到"小规模量产"的关键拐点

2024-2025 年是具身智能（尤其是人形机器人）从 PPT / Demo 走向**小规模量产 + 工业部署**的关键两年：

```
2023: 实验室阶段
  ── RT-2, π₀ 初版, Figure 01, Optimus Gen 1
  ── 单任务 demo、实验室遥操作

2024: 小规模部署
  ── Figure 02 @ BMW, Digit @ GXO/Amazon, Apollo @ Mercedes
  ── 人形双足 BOM 跌破 $50K（Unitree G1 $16K 起步）
  ── π₀ 开源部分权重，跨本体 VLA 成立

2025: 量产拐点
  ── Figure/1X/Optimus 宣布千台级产能规划
  ── 智元 AgiBot World 公开 100 万条真机数据
  ── Helix / GR00T N1 / π₀.5 确立端到端 VLA 共识
  ── 世界模型（Cosmos/Genie/V-JEPA 2）成为"第三条训练腿"

2026 (现在): 数据飞轮 + 商业化验证
  ── 百台级 B 端场景 ROI 验证
  ── 端侧 VLA 芯片（Jetson Thor）量产
  ── 首批 C 端家用 Beta（1X Neo、Figure 03）
  ── 头部公司完成 B+ / C 轮或 IPO 进场
```

### 1.2 B 端先于 C 端 2-3 年

**当前产业共识**：工业/物流 B 端比家庭/消费 C 端领先 2-3 年成熟：

| 维度 | B 端（工业/物流） | C 端（家庭/服务） |
|------|------------------|------------------|
| **场景结构化** | 高（固定夹具、已知物体） | 低（开放世界、长尾物体） |
| **容错率** | 99% 长尾可人工兜底 | 安全零容忍 |
| **成本预算** | $20-50K 可接受（替代 2 人工） | <$20K 才有规模 |
| **法规/保险** | 工业安全标准成熟 | 家用未立法 |
| **2026 状态** | 百台级部署验证 ROI | 首批 Beta，规模化 2027+ |

### 1.3 数据飞轮开始闭环

2026 年头部公司首次出现"真实部署 → 数据回传 → 模型迭代 → 性能提升"的**完整飞轮**：

- **Figure**：BMW 工厂 02 每日生成操作数据 → Helix 迭代
- **智元 AgiBot**：数据工厂 + 商业化机器人回流 → GO 系列迭代
- **Physical Intelligence**：多家伙伴本体数据汇聚 → π 系列跨本体泛化
- **Tesla**：员工试用生成数据 → Optimus 持续训练

**核心判断**：谁先跑通"百台规模真机数据飞轮"，谁就拿到 2027-2028 窗口期的复利优势。
