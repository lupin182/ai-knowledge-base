# 主流 VLA 模型 2026

> 本文盘点 2025-2026 年主流 VLA 模型，按机构分组：Physical Intelligence π 系列、NVIDIA GR00T、Figure Helix、Google DeepMind Gemini Robotics、国内 VLA（RDT-2 / 智元 GO-1 / UniAct / 银河通用 / 自变量机器人）、开源轻量 VLA（OpenVLA / SmolVLA / TinyVLA / CogACT）。每个模型给出参数、动作头、频率、开源状态、本体、发布日期。

## 二、Physical Intelligence π 系列

### 2.1 π₀（2024.10）

**第一个大规模 Flow Matching VLA**，是 PI 公司的奠基作。

- **架构**：PaliGemma-3B 主干（SigLIP-So400m + Gemma-2B）+ Flow Matching Transformer 动作头
- **输入**：2-3 路 RGB 多视角 + 本体状态（proprioception） + 语言指令
- **输出**：50 步动作 chunk（约 1 秒未来）
- **训练数据**：~10,000 小时真实遥操作 + Open X-Embodiment 子集 + 仿真增强
- **训练策略**：
  1. VLM 多模态预训练
  2. 跨本体机器人联合训练
  3. 任务特化微调
- **特点**：
  - 多种本体（单臂、双臂、不同夹爪）
  - 形变物体操作（布料、食物）

### 2.2 π₀.5（2025.04）

面向**家庭开放世界泛化**的重要升级。

- **Co-Training**：网络多模态数据 + 真机 + 仿真联合训练
- **高层语言推理**：模型自主分解长任务（"清理厨房" → 多步子任务）
- **discrete language action**：动作 token 中穿插自然语言规划步
- **定量结果**：
  - 未见过的家庭场景 20-60 分钟长时任务成功率 **40-90%**
  - 物体泛化：训练未见过的 100+ 物体

### 2.3 π-fast（2025 H2）

低延迟轻量版：

- 参数减半
- 推理速度 3-5×
- 保留 π₀.5 主体能力
- 目标 30 Hz+ 闭环
- 适合端侧部署

### 2.4 π₀.6 / π₀.7（2026 H1，据披露）

PI 的最新迭代升级。**重点方向（据披露/预期）**：

- **多模态推理增强**：更深的空间推理与物理常识理解
- **长 horizon**：从 π₀.5 的几十分钟向小时级任务扩展
- **家庭泛化**：跨家、跨物体、跨光照的零样本稳健性
- **可能的世界模型耦合**：用 Cosmos 类模型做 rollout 增广与反事实规划
- **端侧适配版本**：预期有配套的部署友好版本

具体的架构细节、参数规模、数据量以 PI 后续论文/博客为准。

### 2.5 Flow Matching 训练数学回顾

```
训练目标:
  L_FM = E_{t, a_0, a_1} || v_θ(a_t, t, obs) - (a_1 - a_0) ||²
  其中 a_t = (1-t)·a_0 + t·a_1   (线性插值路径)
        a_0 ~ N(0, I)              (标准噪声)
        a_1 为目标动作              (来自真机数据)
        t ~ Uniform[0, 1]

推理:
  a_0 ~ N(0, I)
  for k = 0 to K-1:
      t = k / K
      a_{t + 1/K} = a_t + (1/K) · v_θ(a_t, t, obs)
  return a_1
  (K = 5~10 步即可)
```

详见 [动作生成方法.md § Flow Matching](动作生成方法.md)。

## 三、NVIDIA GR00T 系列

### 3.1 GR00T N1（2025.03，首个 Apache 2.0 开源 VLA）

面向人形机器人的基础模型。

```
架构: System 2 + System 1
  System 2: Eagle-2 VLM (2B)
  System 1: Diffusion Transformer (DiT, 1B)
            输入: VLM token + 本体状态
            输出: 动作 chunk (16 步)

训练数据:
  - Open X-Embodiment
  - AgiBot-World (100 万条)
  - 仿真 (Isaac Sim / GR00T Dreams)
  - Fourier GR-1 / Unitree H1 真实数据

控制频率: VLM 7 Hz / DiT 30+ Hz
开源: 模型权重 + 训练代码 + 数据 pipeline
支持本体: Fourier GR-1、Unitree H1、Neura 4NE-1、1X Neo、ANYbotics ANYmal 等
```

### 3.2 三阶段训练 pipeline

1. **Stage 1**：VLM 预训练（保留 Eagle-2 多模态能力）
2. **Stage 2**：Action Expert 在大规模跨本体数据上训练
3. **Stage 3**：特定本体微调

### 3.3 GR00T N1.5 / N2（2025 H2 - 2026）

预期改进：

- 更大的动作专家，支持更长 horizon
- 触觉输入通道
- 与 Cosmos 世界模型联合训练
- 持续开源

### 3.4 GR00T Dreams

NVIDIA 的**合成数据生成 pipeline**（基于 Cosmos 世界模型）：

- 从少量真实轨迹 → 生成大量变体（新视角、新物体、新场景）
- 用于扩展训练数据规模
- 2025 GTC 宣布公开

> 世界模型详见 [../02-世界模型/](../02-世界模型/)。

## 四、Figure Helix（2025.02）

Figure AI 的首个端到端 VLA，部署在 Figure 02 上。是**双系统双频架构的工程范例**。

### 4.1 架构细节

```
System 2 (S2):
  - 7B VLM (internet-scale 预训练)
  - 9 Hz
  - 输入: 多相机 RGB + 语言
  - 输出: continuous latent vector (256 维)

System 1 (S1):
  - 80M Transformer
  - 200 Hz
  - 输入: S2 latent + 本体状态（关节角、速度、末端力）
  - 输出: 连续动作（全身控制，35-DoF 上半身）

训练:
  - ~500 小时高质量 Figure 02 遥操作
  - 单一策略覆盖: 抓取、放置、递送、开柜门、家居
  - 无需任务特定微调
```

### 4.2 技术亮点

1. **单一神经网络全身协调**：双臂 + 躯干 + 手 + 头，一个模型全输出
2. **双机器人协作**：Helix 支持两台 Figure 02 协同（一递一放）
3. **端到端双频解耦**：不是"VLM 规划 + 技能库"，而是端到端训练的双频模型
4. **商业数据闭环**：BMW 工厂部署持续产生真机数据

### 4.3 Helix v2（2025 H2）

部署 Figure 03。改进方向：

- 更长 horizon 推理
- 家庭场景（洗碗、叠衣）
- 商业化路径从工业向家庭延伸

## 五、Google DeepMind Gemini Robotics（2025.03）

### 5.1 Gemini Robotics

基于 Gemini 2.0 的 VLA，DeepMind 回归机器人领域的旗舰工作。

```
主干: Gemini 2.0 Vision（经机器人数据继续训练）
动作头: Diffusion Transformer
场景: 桌面双臂操控（ALOHA 2 类硬件）
控制频率: 10-30 Hz
```

**关键能力**：

- **Generality**：大量跨物体 / 跨任务泛化
- **Interactivity**：响应自然语言指令（含中途修正）
- **Dexterity**：折纸船、拉拉链、装配乐高

### 5.2 Gemini Robotics-ER（Embodied Reasoning）

ER 版本专注**具身推理**，不直接输出动作：

- 空间理解：物体位置、可供性（affordance）预测
- 长 horizon 规划：分解为子步骤
- 代码生成：生成控制代码（类似 Code as Policies）

**思路**：VLM 做大脑，接任何动作后端（策略网络 / 经典控制 / 第三方 VLA）都可用。可视为"**动作中立的上半身**"。

### 5.3 DeepMind × Apptronik 合作

2025 年 DeepMind 与 Apptronik 深度合作：

- Gemini Robotics 在 **Apollo** 人形上部署
- 共享研究成果
- 对标 Figure + OpenAI 的组合

## 六、国内 VLA

### 6.1 RDT-2（清华 / 星动，2025-2026）

**Robotics Diffusion Transformer 第二代**：

- 基于 SigLIP + T5
- **1B 参数**，双臂 / 双机协作
- **MoE 动作头**：不同任务激活不同专家
- 开源，国内最强 VLA 之一

### 6.2 智元机器人 GO-1 / UniAct（2025）

**GO-1**：智元开源大规模 VLA，在 AgiBot World 上训练，面向远征/灵犀本体。

**UniAct**：智元的"全身动作"统一接口，把上肢操控 + 下肢移动/行走合并到同一动作空间。

### 6.3 银河通用 Galbot（2024-2025）

- 本体：Galbot G1（双臂 + 轮式）
- VLA 闭源，面向商超、零售场景
- 走"大模型 + 双臂移动"路径

### 6.4 自变量机器人（2025）

- 自研双臂本体
- VLA 闭源
- 聚焦工业装配与家庭场景

### 6.5 国内其他

- **Fourier GR-1 / GR-2**（傅利叶）：硬件为主，GR00T 的参考本体之一
- **宇树**：Unitree H1 / G1 硬件 + 开源控制栈，VLA 生态依赖外部
- **星海图、穹彻智能、乐聚、开普勒**：不同程度的自研 VLA 与遥操作数据采集

## 七、开源 / 轻量 VLA

### 7.1 OpenVLA（2024.06）

- 架构：SigLIP + DinoV2 + Llama 2 7B
- 动作：离散 token
- 数据：Open X-Embodiment 子集（97 万轨迹）
- **完全开源**：权重、代码、训练流程

### 7.2 OpenVLA-OFT（2024.10）

OpenVLA 的优化微调版：

- **并行解码**：26 Hz → 109 Hz
- **L1 回归代替 cross-entropy**
- **Action chunking**
- 证明 OpenVLA 的瓶颈在离散 token 与自回归，而非模型能力

### 7.3 SmolVLA（2025.05，HF LeRobot）

- **~500M 参数**
- Flow Matching 动作头
- 端侧部署（Jetson Orin / Mac Mini 可跑）
- 面向 HF LeRobot 教育生态

### 7.4 TinyVLA（2025）

- <1B 参数
- Diffusion 动作头
- 桌面场景
- 降低研究门槛

### 7.5 CogACT（微软亚研，2024.11）

- VLA 中解耦认知（cognition）和动作（action）
- VLM 编码器 + 独立的 diffusion action decoder
- 30 Hz，SOTA 级成功率
- 开源

### 7.6 其他

- **Dexbotic（2024）**：灵巧操作 VLA
- **iDP3**（Improved 3D Diffusion Policy）：3D 输入版 Diffusion Policy
- **SpiRit-LM**：多模态生成 + 动作
