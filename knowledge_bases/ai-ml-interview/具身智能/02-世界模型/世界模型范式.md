# 世界模型范式

> 世界模型（World Model, WM）是智能体对环境动力学的内部表示，记作 $P(s'|s,a)$ 或 $P(o'|o,a)$。本文梳理为何具身智能需要世界模型、它的三大价值，以及像素 / 隐空间 / 可交互三种范式与 Dreamer 系列谱系。

## 一、为什么需要世界模型

### 1.1 策略直接映射 vs 世界模型中 rollout

传统强化学习与模仿学习直接学策略：

$$\pi_\theta(a \mid s) \approx \pi^*(a \mid s)$$

**问题**：
- 样本效率低，每个新任务都要大量真机/仿真交互
- 无反事实推理能力（"如果我左转而不是右转会怎样？"）
- 策略与环境紧耦合，换任务就要重训

**引入世界模型**后，智能体学两个部分：

$$\underbrace{P_\phi(s_{t+1} \mid s_t, a_t)}_{\text{世界模型}} \qquad \underbrace{\pi_\theta(a_t \mid s_t)}_{\text{策略}}$$

策略可以在"想象"中 rollout，无需真实交互即可评估动作序列：

```
真实交互:    s_0 →[π]→ a_0 →[env]→ s_1 →[π]→ a_1 → ...
想象 rollout: s_0 →[π]→ a_0 →[WM]→ ŝ_1 →[π]→ a_1 → ...
                                    ↑
                              不消耗真实数据
```

### 1.2 对具身智能的特殊价值

机器人领域真机数据稀缺（见 [../../具身智能/数据与训练范式演进.md](../数据与训练范式演进.md) 第一节），世界模型的价值被进一步放大：

- **真实采样代价极高**：每条遥操作轨迹 10-60 秒人类劳动
- **探索危险**：真机 RL 容易损坏硬件
- **分布多样性不足**：一个实验室环境有限

世界模型把"采样一条轨迹"从"物理世界一次交互"降为"GPU 上的一次前向传播"。

## 二、世界模型的三大价值

### 2.1 数据增广（Data Augmentation）

```
少量真实轨迹 → 训练 / 调用 WM → 大量变体轨迹

典型增广维度:
  - 光照 (白天/夜晚/灯光方向)
  - 物体 (替换为同类别不同实例)
  - 背景 (换房间/换桌面)
  - 视角 (相机位姿扰动)
  - 干扰物 (添加/移除无关物体)
```

代表工作：NVIDIA **GR00T Dreams**（基于 Cosmos）将真机采集的 1000 条轨迹扩展到 10 万条视觉变体，显著提升 VLA 的视觉鲁棒性。

### 2.2 规划搜索（Planning）

```
给定目标 g (可以是语言/图像/隐表示)
在 WM 中展开动作序列 a_{0:T}
评估 cost(ŝ_T, g) 或 cumulative reward
用 CEM / MPPI / 梯度 选出最优动作
```

经典做法：**Dreamer V3** 在隐空间中用 Actor-Critic 学策略；**V-JEPA 2** 直接在隐空间做 MPC 式规划。

**优势**：
- 可以做反事实推理
- 面对新任务（新目标图像）可以 zero-shot 规划
- 不需要为每个任务训练新策略

### 2.3 无监督学习（Unsupervised Representation Learning）

世界模型的训练目标 $P(s_{t+1}|s_t, a_t)$ 本身是**自监督**的，不需要人工标注：

```
大量无标签视频 (YouTube / Ego4D) → 训练 WM
→ WM 内部的 encoder 自动学到物理规律、物体恒常性、因果关系
→ 可以作为下游 VLA 的视觉预训练
```

代表：**VIP**、**R3M**、**V-JEPA 2** 的 encoder 被广泛用作下游机器人策略的视觉 backbone。

## 三、三类世界模型范式

### 3.1 像素世界模型（Pixel-Space WM）

**直接在像素空间预测下一帧**：

$$P(o_{t+1} \mid o_{\le t}, a_{\le t}) \text{，其中 } o_t \in \mathbb{R}^{H \times W \times 3}$$

```
代表:
  - NVIDIA Cosmos Predict (5B / 14B 扩散 / 12B 自回归)
  - DeepMind Genie 1/2/3
  - UniSim / Sora（非具身导向但同源）
  - 1X World Model

架构:
  视频扩散模型 (DiT) 或 时空自回归 Transformer

优势:
  - 可视化直观（可以直接"看"预测）
  - 保留完整视觉信息
  - 可以直接作为数据增广

劣势:
  - 算力开销大（预测全分辨率视频）
  - 难以长 horizon 保持一致性
  - 动作生成维度与像素生成不匹配
```

### 3.2 隐空间世界模型（Latent-Space WM）

**先将观测 encode 到低维隐空间，再在隐空间预测**：

$$z_t = f_\text{enc}(o_t), \qquad P(z_{t+1} \mid z_{\le t}, a_{\le t})$$

```
代表:
  - Dreamer 系列 (V1/V2/V3)
  - PlaNet
  - Meta V-JEPA 2
  - IRIS

架构:
  VAE / RSSM（Recurrent State-Space Model）/ JEPA

优势:
  - 算力小 1-2 个数量级
  - 易于长 horizon（隐空间抽象）
  - 规划效率高（CEM / 梯度在低维空间）

劣势:
  - 隐表示无法直接可视化
  - 数据增广能力弱（无法给下游提供图像）
  - 依赖 encoder 质量
```

### 3.3 可交互世界模型（Interactive / Action-Conditioned WM）

介于两者之间，**以动作/控制信号为条件生成可交互画面**，使 WM 更像"虚拟仿真器"：

```
代表:
  - DeepMind Genie 1 (2024): 从无标签视频学 "方向键 → 下一帧"
  - Genie 2 (2024.12): 3D 场景，支持人物/物体交互
  - Genie 3 (2025.08): 更长一致性 + 物理约束

架构:
  Spatial-Temporal Transformer + Latent Action Model

优势:
  - 可以作为"无限生成式仿真器"
  - 交互式数据采集（策略与 WM 实时互动）

劣势:
  - 物理正确性无保证
  - 分辨率与帧率受限
```

### 3.4 三类对比

| 维度 | 像素 WM | 隐空间 WM | 可交互 WM |
|------|---------|-----------|-----------|
| **主要代表** | Cosmos, 1X WM | Dreamer V3, V-JEPA 2 | Genie 系列 |
| **算力开销** | 高 | 低 | 中 |
| **主要用途** | 数据增广、Sim2Real | 规划、表示学习 | 生成式仿真 |
| **可视化** | ✅ 直接图像 | ❌ 需解码 | ✅ 直接图像 |
| **长 horizon** | 中（1-2 分钟） | 强（抽象后稳定） | 中（Genie 3 推至分钟级） |
| **动作接口** | 条件控制 | 显式 a_t | 键盘/手柄式 |

## 四、Dreamer 系列谱系

Dreamer（Hafner 等人，2019 起）是隐空间 WM + RL 的里程碑系列，奠定了现代世界模型的 RSSM 架构。

### 4.1 Dreamer V1（2019）

> Hafner et al., "Dream to Control: Learning Behaviors by Latent Imagination", 2019.

**核心架构：RSSM（Recurrent State-Space Model）**

```
RSSM 状态: s_t = (h_t, z_t)
  h_t: 确定性 RNN 隐状态
  z_t: 随机隐变量 (捕获不确定性)

三个组件:
  Representation:  z_t ~ p(z_t | h_t, o_t)     (后验, 编码观测)
  Transition:      ẑ_t ~ p(z_t | h_t)          (先验, 不看观测预测)
  Recurrent:       h_t = f(h_{t-1}, z_{t-1}, a_{t-1})

训练损失:
  重建: log p(o_t | s_t)
  KL:   KL( q(z_t | h_t, o_t) || p(z_t | h_t) )
  奖励: log p(r_t | s_t)

策略训练:
  在想象中 rollout 15 步 → Actor-Critic 学策略
  Actor 和 Critic 都只看 s_t，不看原始像素
```

**成就**：在 DeepMind Control Suite 上样本效率超过当时所有基线。

### 4.2 Dreamer V2（2020）

> Hafner et al., "Mastering Atari with Discrete World Models", 2020.

**关键改变**：
- 隐变量从连续高斯改为**离散 one-hot**（32 类 × 32 维）
- 更稳定，更适合像素复杂的 Atari
- 首个在 Atari 上达到人类水平且样本效率高的 model-based 方法

### 4.3 Dreamer V3（2023）

> Hafner et al., "Mastering Diverse Domains through World Models", 2023.

**里程碑**：**固定超参数**在 150 多个不同任务上都能工作，包括：
- Atari、DMControl、ProcGen
- Minecraft（从零开始挖钻石，用时约 20K env steps）
- 4 足机器人真机训练（见下文 DayDreamer）

**关键技术**：
- symlog 归一化（适配不同尺度的奖励）
- 两个 critic + 指数移动平均 target
- 离散隐变量 + KL balancing

**意义**：Dreamer V3 是**第一个真正通用的 model-based RL 算法**，成为后续世界模型工作的基准。

### 4.4 DayDreamer（2022）

> Wu et al., "DayDreamer: World Models for Physical Robot Learning", 2022.

**首次将 Dreamer 部署到真实机器人**：
- 4 足 A1 行走：1 小时真机训练从零学会
- UR5 拾取：10 小时学会抓小物体
- XArm 推球、Sphero 导航

**方法论价值**：证明 model-based RL 在真机上样本效率远高于 model-free（PPO / SAC）。

### 4.5 IRIS（2023）

> Micheli et al., "Transformers are Sample-Efficient World Models", 2023.

**用 Transformer 替代 RSSM**：
- VQ-VAE 把图像 tokenize
- Causal Transformer 做 token 级下一帧预测
- Atari 100K 上 SOTA

**意义**：打开"Transformer WM"路线，后续 Cosmos Autoregressive、Genie 都沿此思路。

## 五、世界模型与具身智能的关系演进

### 5.1 2018-2022：RL 加速器

```
定位: WM 是 model-based RL 的"仿真器"
价值: 样本效率提升 10-100 倍
局限: 隐空间表示与像素可视化分离，只服务 RL
```

### 5.2 2023-2024：生成式仿真

```
定位: WM 是从视频学习的"通用仿真器"
价值: 从无标签视频学物理常识
代表: UniSim, Genie 1
局限: 还没真正被具身领域用于训练 VLA
```

### 5.3 2025-2026：世界基础模型（WFM）

```
定位: WM 与 VLM、VLA 并列为"机器人三大基础模型"
价值: 
  - 数据增广（取代部分仿真）
  - Sim2Real 桥梁
  - 与 VLA 联合训练
代表: Cosmos, V-JEPA 2, Genie 3
```

详见 [主流世界模型 2026](主流世界模型2026.md) 与 [VLA 与世界模型融合](VLA与世界模型融合.md)。
