# 07 · 数据与遥操作

> "具身智能的瓶颈是数据，不是模型。"—— 2025-2026 行业主流判断。本专题梳理数据源、公开数据集、数据工厂 / 飞轮与跨本体学习。

## 为什么数据是瓶颈

```
LLM vs 具身智能的数据规模对比：

LLM (GPT-4):
  预训练 ~10 万亿 tokens（整个互联网）
  RLHF  数十万条人类偏好

具身智能 (π₀ 训练集):
  真机数据 ~1 万小时（~数百万条轨迹）
  ≈ 2-3 年全人类机器人数据总量

差距：5-6 个数量级
```

### 数据采集的四大难点

1. **需要物理接触**：不能从网页爬，必须真机采集
2. **跨本体不通用**：Franka 数据不能直接用于 UR5（夹爪、关节结构差异）
3. **任务特异性**：叠衣服数据对做饭意义不大
4. **质量要求高**：失败轨迹需要筛选，专家水平遥操作稀缺

## 2026 数据发展现状

### 时间线

```
2017-2020  单任务采集，规模千条级
           BC-Z, RoboNet 首次尝试跨任务

2022       RT-1 (Google) 13 万真实数据，Transformer 初露锋芒
           Ego4D (Meta) 3670 小时人类第一视角

2023       RT-2 将 VLM 引入动作学习
           Open X-Embodiment (OXE)：21 机构联合，100 万+ 轨迹，22 种机器人
           BridgeData V2 (Berkeley) 60K 轨迹

2024       DROID 76K 轨迹 / 564 场景（分布式采集）
           AgiBot World (智元) 100 万+ 真机轨迹，217 任务
           Mobile ALOHA 普及家庭任务
           Apple Vision Pro / Quest 3 遥操作兴起
           数据工厂概念成型（智元张江、Tesla 训练中心）

2025       五源融合训练（真机+仿真+世界模型+人类视频+自主策略）
           世界模型（Cosmos）作为"无限数据"候选
           RLHF 类偏好数据首次在具身领域试水（RoboArena）
           小时级数据飞轮仍未真正跑通

2026 (Q1-Q2)
           数据工厂进入规模化竞赛
           OXE v2、AgiBot World v2 等扩展集陆续出现
           VLA + RL 后微调成为标配
```

### 当前共识

1. **五源融合训练是基本范式**：真机 + 仿真 + 世界模型 + 人类视频 + 自主策略
2. **数据是瓶颈，模型架构相对收敛**：Transformer 主干 + Flow Matching / Diffusion 动作头
3. **跨本体是必要方向**：Open X-Embodiment 验证了跨本体能带来零样本迁移
4. **数据工厂军备竞赛**：智元、Tesla、Physical Intelligence、Figure、银河通用、1X 等投入数百到数千遥操作工位
5. **仿真 + 真机混合训练优于纯仿真**：纯仿真 VLA 真机性能通常差 20-40%
6. **遥操作仍是主力**：Apple Vision Pro + Open-TeleVision 成为 2025 新热点

### 开放问题

- **小时级数据飞轮尚未跑通**：部署 → 反馈 → 筛选 → 再训练 的闭环仍在 Alpha/Beta
- **合成数据 vs 真实数据之争**：银河通用 / Genesis 代表合成派，智元 / Figure 代表真实派
- **跨本体极限**：humanoid 和机械臂是否应该混训仍有争议
- **数据质量标注缺标准**：失败原因、成功判据、任务描述无统一规范
- **隐私与安全**：家庭场景数据回传涉及法律合规
- **偏好数据的获取与利用**：RLHF 在机器人领域仍是开放课题
