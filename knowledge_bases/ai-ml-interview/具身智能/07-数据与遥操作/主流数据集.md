# 主流数据集

> 本文汇总 2026 具身智能主要公开数据集：真机轨迹、多机构合作集、人类视频、仿真集，以及 2025-2026 的新增扩展。

## 二、Open X-Embodiment (OXE) 详解

### 2.1 基本信息

- **发布**：2023.10，21 家机构联合（Google DeepMind 主导）
- **规模**：100 万+ 真实机器人轨迹
- **机器人类型**：22 种（Franka, UR5, Sawyer, WidowX, Kuka, Google Robot 等）
- **任务**：527 项（抓取、推、放置、工具使用、导航）

### 2.2 意义

- **跨本体学习**的里程碑
- 证明多机构联合可行
- 推动社区形成**共享数据**文化
- 基于 OXE 训练的 RT-X 验证了跨本体迁移

### 2.3 数据格式

- **RLDS (Reinforcement Learning Datasets Format)**
- TFRecord 存储
- 每条轨迹：(obs_sequence, action_sequence, language, metadata)

### 2.4 OXE v2（2025 扩展）

- 规模扩展至 200 万+
- 加入 2024 新贡献（π₀ 伙伴数据）
- 加入人形数据（GR00T 贡献）
- 任务描述质量提升（LLM 辅助重写）

## 三、AgiBot World 详解

### 3.1 基本信息

- **发布**：2024.12，智元机器人
- **规模**：100 万+ 真机轨迹
- **任务**：217 种，覆盖家居、餐饮、办公、零售
- **硬件**：远征 A2（双臂）
- **采集**：上海张江数据工厂
- **开源协议**：CC BY-NC 4.0

### 3.2 数据工厂背景

- 100+ 工位
- 24 小时三班制
- 单工位日产 50-200 条
- 预计年产能 300 万+

### 3.3 配套模型

- **GO-1**（智元 VLA）：基于 AgiBot World 训练
- 开源 7B 规模

### 3.4 AgiBot World v2（2025）

- 规模扩展至 200 万+
- 加入人形本体数据
- 细化任务分类（长 horizon、多步骤）
- 与 π₀ / GR00T 等合作扩展

### 3.5 意义

- 中国首个 100 万级开源真机数据集
- 改变"高质量数据只在美国"的格局
- 成为 π₀ 等海外模型的补充训练资源

---

## 四、DROID

### 4.1 基本信息

- **发布**：2024，Stanford / Berkeley / Toyota 等联合
- **规模**：76K 轨迹
- **场景**：564 个不同场景（分布式采集）
- **机器人**：Franka Panda
- **特点**：**场景多样性**是最大卖点，分布于各高校和家庭

### 4.2 采集方式

- 分布式采集：多个实验室独立采集
- 统一硬件 + 统一工具链
- 每个采集点采集本地场景（厨房、书房、办公室等）

### 4.3 意义

- 证明**分布式采集**的可行性
- 场景多样性对泛化至关重要
- OXE 的重要贡献者

---

## 五、BridgeData V2

### 5.1 基本信息

- **发布**：2023，Berkeley（Sergey Levine 组）
- **规模**：60K 轨迹
- **机器人**：WidowX 250
- **任务**：多任务操作，覆盖日常家务
- **特点**：已成为经典基准

### 5.2 意义

- 模仿学习早期的重要基准
- π₀ / OpenVLA / Octo 等模型都在此评测
- 小规模但质量高，适合学术验证

---

## 六、RT-X

### 6.1 基本信息

- **发布**：2023，Google DeepMind
- **规模**：500 万+ steps（注意：steps 而非轨迹）
- **来源**：OXE + Google 内部数据
- **模型**：RT-X 系列（RT-1-X, RT-2-X）

### 6.2 与 OXE 关系

- RT-X 是**模型**，OXE 是**数据集**
- RT-X 在 OXE 上训练
- 发布时两者一起，常被混用

---

## 七、LIBERO

### 7.1 基本信息

- **发布**：2023，UT Austin
- **规模**：130 任务，多难度
- **类型**：**仿真** benchmark（基于 robosuite/MuJoCo）
- **特点**：**长 horizon** + **终身学习**评测

### 7.2 任务划分

- **LIBERO-Spatial**：空间推理
- **LIBERO-Object**：不同物体
- **LIBERO-Goal**：不同目标
- **LIBERO-10**：10 个长 horizon 任务
- **LIBERO-90**：大规模评测

### 7.3 意义

- VLA 模型长 horizon 评测标配
- 仿真 + 统一评测，便于对比

---

## 八、RoboSet、RH20T 等

### RoboSet

- **来源**：Meta / CMU
- **规模**：100K 轨迹
- **机器人**：Franka
- **特点**：多任务、多场景

### RH20T

- **来源**：上海交大（卢策吾组）
- **规模**：147K 轨迹
- **任务**：147 项，覆盖工具使用、双手协作
- **特点**：中国最早的大规模真机数据集之一

---

## 九、人类视频数据集

### 9.1 Ego4D（Meta, 2022）

- **规模**：3670 小时第一视角
- **来源**：9 国 13 机构
- **标注**：叙述、动作类别、手-物体交互
- **用途**：视觉 encoder 预训练（R3M、MVP）、动作学习

### 9.2 Ego-Exo4D（Meta, 2024）

- **规模**：5625 视角对（第一+第三视角同步）
- **特点**：同一动作的第一视角 + 第三视角配对
- **用途**：视角迁移、动作理解、人类技能学习

### 9.3 Something-Something、EPIC-KITCHENS

- Something-Something：基础动作理解
- EPIC-KITCHENS：厨房第一视角
- 用于视觉预训练、动作分类

### 9.4 YouTube 操作教程

- 非正式数据源，但规模巨大
- 用于弱监督预训练
- π₀.5 声明使用"大量多模态互联网数据"

---

## 十、仿真合成数据集

### RoboCasa Dataset（UT Austin, 2024）

- 100K+ 条家庭场景仿真轨迹
- MuJoCo / robosuite
- 用于仿真预训练 + Sim-to-Real

### RoboTwin Dataset（上海交大, 2024）

- 20K+ 双臂任务
- LLM 生成脚本 → SAPIEN 仿真执行
- 代表"LLM + 仿真"合成路径

### SIMPLER Benchmark（2024）

- 不是数据集，是评测工具
- 校准仿真评测与真实评测
- 用于 VLA 模型（RT-2 等）对齐

## 十二、数据集选择指南

### 学术研究入门

```
BridgeData V2 (经典、小规模)
  + LIBERO (仿真 benchmark)
  + 自采 100-500 条
→ 训练 ACT / Diffusion Policy / OpenVLA 微调
```

### 工业预训练

```
OXE + AgiBot World (真机主力)
  + RT-X / RH20T (扩展)
  + 仿真 (Isaac / Genesis 千万级)
  + 世界模型增广 (可选)
→ 训练自研 VLA
```

### 跨本体研究

```
OXE (22 本体)
  + 自有本体数据
→ 训练 Octo / π₀ 类模型
→ 跨本体零样本评测
```

### 人类视频预训练

```
Ego4D / Ego-Exo4D
→ 预训练视觉 encoder
→ 用于 VLA 初始化
```

### 长 horizon 评测

```
LIBERO-10 / LIBERO-90
  + 自研长任务
→ 评测模型长序列能力
```

## 十三、2025-2026 新增与合作扩展

### OXE v2

- 规模：200 万+
- 新贡献：π₀ 合作伙伴数据、GR00T 人形数据
- 任务标注：LLM 辅助重写，质量提升
- 时间：预计 2025 下半年 ~ 2026 上半年

### AgiBot World v2

- 规模：200 万+
- 加入人形本体（远征 / 灵犀）
- 细化长 horizon 任务

### RoboArena（评测）

- 真机跨本体评测平台
- 多机构参与，类似 LLM Arena
- 2025 年首次发布

### 社区趋势

- **合作 > 独占**：2025 起数据分享成为新常态
- **HuggingFace 数据集 hub**：LeRobot 格式成为社区默认
- **隐私合规**：家庭数据采集开始涉及 GDPR / PIPL
