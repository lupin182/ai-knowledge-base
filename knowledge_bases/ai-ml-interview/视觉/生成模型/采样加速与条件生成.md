# 采样加速与条件生成

> DDPM 需要 $T=1000$ 步采样，太慢。本文覆盖主要加速方法和条件生成技术。

---

## 1. DDIM（Denoising Diffusion Implicit Models）

Song et al. (2020) 的核心发现：DDPM 的训练目标（预测噪声）其实**不依赖于马尔可夫假设**。可以定义**非马尔可夫**的前向过程，使得可以跳步采样。

### 采样公式

$$x_{t-1} = \sqrt{\bar{\alpha}_{t-1}} \underbrace{\left(\frac{x_t - \sqrt{1-\bar{\alpha}_t} \cdot \epsilon_\theta}{\sqrt{\bar{\alpha}_t}}\right)}_{\text{预测的 } x_0} + \sqrt{1 - \bar{\alpha}_{t-1} - \sigma_t^2} \cdot \epsilon_\theta + \sigma_t \cdot z$$

关键参数 $\sigma_t$ 控制随机性：
- $\sigma_t = \tilde{\beta}_t^{1/2}$（DDPM 的值）：退化为标准 DDPM 采样
- $\sigma_t = 0$：采样变成**完全确定性的**

### 跳步采样

$\sigma_t = 0$ 时，可以大幅跳步而不严重影响质量：

```
DDPM:  x₁₀₀₀ → x₉₉₉ → x₉₉₈ → ... → x₁ → x₀   (1000 步)
DDIM:  x₁₀₀₀ → x₉₀₀ → x₈₀₀ → ... → x₁₀₀ → x₀  (10 步，质量略降)
DDIM:  x₁₀₀₀ → x₈₀₀ → x₆₀₀ → x₄₀₀ → x₂₀₀ → x₀ (5 步，质量下降明显)
```

### DDIM 的额外好处

1. **确定性生成**：相同的 $x_T$ 和参数总是生成相同的 $x_0$（可复现）
2. **语义插值**：在 $x_T$ 空间做线性插值，生成结果语义上也平滑过渡
3. **DDIM Inversion**：可以把真实图片"反推"回噪声，用于图片编辑

## 2. Classifier Guidance

Dhariwal & Nichol (2021)：用一个**预训练分类器**的梯度来引导生成。

$$\hat{\epsilon}_\theta = \epsilon_\theta(x_t, t) - s \cdot \sqrt{1-\bar{\alpha}_t} \cdot \nabla_{x_t} \log p_\phi(y|x_t)$$

- $p_\phi(y|x_t)$：在噪声图上训练的分类器
- $s$：guidance scale，控制条件强度

**缺点**：需要额外训练一个噪声分类器，麻烦且会引入分类器的偏差。

---

## 3. Classifier-Free Guidance（CFG）

Ho & Salimans (2022)：不需要额外分类器，用**同一个模型**同时学有条件和无条件生成。

### 训练

训练时以一定概率（如 10%）随机**丢弃条件**（用空 token $\varnothing$ 替代）：

```python
# 伪代码
if random() < 0.1:
    c = null_token  # 无条件
else:
    c = text_embedding  # 有条件
loss = ||ε - ε_θ(x_t, t, c)||²
```

### 推理

加权组合有条件和无条件预测：

$$\bar{\epsilon}_\theta = (1 + w) \cdot \epsilon_\theta(x_t, t, c) - w \cdot \epsilon_\theta(x_t, t, \varnothing)$$

$$= \epsilon_\theta(x_t, t, \varnothing) + (1+w) \cdot \underbrace{[\epsilon_\theta(x_t, t, c) - \epsilon_\theta(x_t, t, \varnothing)]}_{\text{条件方向}}$$

直觉：先做无条件预测，再沿着"条件方向"多走 $w$ 步。

```
w=0:    纯无条件生成（忽略 prompt）
w=1:    标准条件生成
w=7.5:  Stable Diffusion 默认，质量最佳
w=20:   高度贴合 prompt，但过饱和、多样性低
```

### CFG 的代价

每步需要两次前向（有条件 + 无条件），推理速度减半。实际实现中常用 batch 并行两次前向。

## 4. Consistency Models

Song et al. (2023)：训练模型从任意噪声水平 $x_t$ **一步直接映射到** $x_0$，不需要逐步去噪。

### 核心思想

定义一致性函数 $f_\theta$：对于同一条 ODE 轨迹上的所有点 $x_t$，$f_\theta(x_t, t)$ 都应该映射到同一个 $x_0$。

```
ODE 轨迹:  x_T → x_{T/2} → x_{T/4} → ... → x_0

一致性要求:
  f_θ(x_T, T) = f_θ(x_{T/2}, T/2) = f_θ(x_{T/4}, T/4) = x_0

→ 训练好后，直接 f_θ(x_T, T) = x_0，一步生成！
```

### 两种训练方式

- **Consistency Distillation**：从预训练的 Diffusion 模型蒸馏
- **Consistency Training**：直接从数据训练（不需要预训练模型）

---

## 5. 其他加速方法

| 方法 | 思路 | 步数 |
|------|------|------|
| DDIM | 非马尔可夫跳步 | 10-50 步 |
| DPM-Solver | 高阶 ODE 求解器 | 10-20 步 |
| Progressive Distillation | 蒸馏：每轮减半步数 | 2-4 步 |
| Consistency Models | 一致性映射 | 1-2 步 |
| LCM (Latent Consistency) | Consistency + 潜在空间 | 2-4 步 |
