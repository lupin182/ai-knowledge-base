知识库

每个知识库 = knowledge_bases/<slug>/ 下独立的一个文件夹，互不干扰。URL 形如 /kb/<slug>/。

＋ 新建一个空知识库
只能用字母 / 数字 / _ / -，决定 /kb/<这里>/ 的 URL。

知识库 AI 提示词

为某个知识库单独给 AI 助手追加一段提示词（拼在基础系统提示词后面）。例如告诉它该 KB 的写作风格、术语习惯、引用规范等。留空 = 只用基础提示词。新开对话时生效。

AI 后端

Claude CLI 需要本机安装 @anthropic-ai/claude-code 并 claude 登录；自定义 API 填 接口地址 + key + 模型名 即可（DeepSeek / 通义千问 / Kimi / 本地 vLLM·Ollama 等主流 Chat Completions 服务都行）。
